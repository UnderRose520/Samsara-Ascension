# 水墨像素融合风角色原型

## 目标
用现有 `generate_2d_sprites.py` 框架，生成一个64px水墨像素融合风的修仙角色剪影，带金色经脉线。

## 技术方案

### 1. 在 `generate_2d_sprites.py` 中新增函数

新增 `gen_ink_pixel_player()` 函数，使用现有的 `_draw_pixel_sprite()` + ASCII art 模板系统。

### 2. 色板设计（玄玉鎏金体系）

```python
ink_palette = {
    "K": hex_to_rgba("#0a0f0f"),  # 墨玄 - 角色剪影主体
    "D": hex_to_rgba("#1a2428"),  # 淡墨 - 剪影边缘/暗部
    "G": hex_to_rgba("#c4a86a"),  # 冷金 - 经脉线、高光
    "J": hex_to_rgba("#3d7a6e"),  # 玉青 - 灵力流动
    "B": hex_to_rgba("#5b8cce"),  # 雷蓝 - 法术元素
    "R": hex_to_rgba("#d4743e"),  # 炎橙 - 危机/武器
    "W": hex_to_rgba("#d8e8f0"),  # 霜白 - 冰元素、眼睛
    "T": hex_to_rgba("#2a3638"),  # 雾灰 - 半透明过渡
}
```

### 3. ASCII 模板设计（16×16，4x缩放=64px）

角色设计要点（持剑站姿）：
- **轮廓**：墨色剪影，不画五官，靠身形辨识
- **姿态**：右手持剑斜指前方，左手掐剑诀于胸前，衣袂向左飘动
- **经脉线**：金色(G)从丹田沿脊柱向上至头顶，双臂各一条，形成"灵气运转"视觉
- **灵气**：玉青色(J)在剑尖和手部流动，表示灵力灌注
- **飘带**：用雾灰色(T)做衣袂飘带，增加动感
- **剑**：炎橙色(R)剑身+冷金(G)剑尖，表示法器品质

### 4. 输出文件

- `game/assets/sprites/player_ink_pixel_64.png` - 64×64 主sprite
- 同时生成一个 `player_ink_pixel_128.png` 变体（2x=128px）供UI预览

### 5. 验证方式

1. 运行 `python game/tools/generate_2d_sprites.py` 生成sprite
2. 在 Godot 中查看sprite，确认：
   - 剪影轮廓清晰可辨
   - 金色经脉线在暗色背景上醒目
   - 整体风格与现有UI（玄玉鎏金）协调
3. 可选：将sprite放入测试场景，与现有地图背景对比

## 关键文件

- `game/tools/generate_2d_sprites.py` - 新增函数
- `game/assets/sprites/player_ink_pixel_64.png` - 输出
- `game/assets/sprites/player_ink_pixel_128.png` - 输出

## 风险

- 16×16 ASCII分辨率下经脉线细节有限，需要精心设计模板
- 如果效果不理想，可尝试32×32模板（2x缩放=64px）增加细节
