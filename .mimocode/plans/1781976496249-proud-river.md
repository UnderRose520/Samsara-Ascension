# 《轮回仙途》全新 UI 设计方案 — 水墨暗色 × 高饱和粒子

> 本文档完全忽略现有 UI 实现，从零设计一套全新的视觉系统。
> 设计方向：**水墨仙侠 × 暗色深渊 × 高饱和粒子爆发**

---

## 一、设计宣言 (Design Manifesto)

**「墨入深渊，粒子绽道」**

这套 UI 的灵魂是**墨与光的对抗**。底色是无尽的黑——不是纯黑，而是带着微弱靛蓝和深紫的墨色深渊，如同修士闭关时眼前的混沌。在这片黑暗之上，**水墨笔触**勾勒出面板的骨架——不是规整的矩形边框，而是毛笔一挥而就的墨痕，起笔重、收笔轻、转折处有飞白。

而**高饱和粒子**是这套 UI 的呼吸。当玩家选择一个仙品词条时，紫色光粒子如萤火般从卡片中涌出；当连击达到百次时，整个屏幕被元素色的墨浪吞没；当道统觉醒时，金色书法文字逐笔写出，每一笔都拖着燃烧的粒子尾迹。

**核心美学公式：** `近黑底色 + 水墨笔触骨架 + 高饱和粒子灵魂 + 金线点缀`

## 三、字体系统 (Typography)

### 3.1 字体层级

| 层级 | 中文 | 拉丁/数字 | 字号 (1080p) | 字重 | 效果 |
|------|------|-----------|-------------|------|------|
| Display | 方正清刻本悦宋 / 汉仪尚巍手书 | — | 36-48px | Bold | 标题用，金色描边1px + 墨色投影 |
| Title | 思源宋体 CN Bold | — | 24-28px | Bold | 面板标题 |
| Body | 思源黑体 CN Regular | Inter | 14-16px | Regular | 正文描述 |
| Caption | 思源黑体 CN Light | Inter Light | 11-13px | Light | 辅助说明 |
| Number | JetBrains Mono | Mono | 16-24px | Bold | 伤害数字、冷却、灵石 |
| Calligraphy | 自制水墨书法体 | — | 28-60px | — | 道统觉醒、天机一击等仪式文字 |

### 3.2 排版规则

- 标题文字使用 **宋体 + 金色描边 1px + 墨色投影 2px**，强化仙侠感
- 伤害数字使用 **等宽粗体**，颜色按类型：白(普通)/金(暴击)/红(斩杀)/紫(混沌)/虹(万法归一)
- 面板内文字行距 = 1.5x 字号，段间距 = 0.8x 字号
- 中文标点使用全角，数字和英文使用半角
- 重要数值使用 `text.gold` 色 + 微弱脉冲动画

## 四、图标系统 (Iconography)

### 4.1 图标风格定义

**「水墨线描 × 元素光晕」**

- **线条：** 所有图标使用 **2px 水墨笔触线条**（非均匀粗细，起笔粗收笔细），模拟毛笔勾勒
- **填充：** 图标内部使用元素高饱和色的 **半透明渐变填充**（opacity 30-60%）
- **光晕：** 图标外围 2-3px 同色柔光（glow），在深色背景上如同墨纸上滴了一滴彩墨
- **尺寸规范：** 16px(微型) / 24px(小型) / 32px(标准) / 48px(中型) / 64px(大型) / 96px(特大)

### 4.2 元素图标设计概念

| 元素 | 造型概念 | 笔触描述 |
|------|---------|---------|
| 火 | 一团向上的火焰，中心最亮 | 起笔粗重（火焰底部），向上逐渐变细，顶端散开为飞白 |
| 水 | 三道波纹涟漪 | 流畅的弧线，起笔轻收笔重，如行云流水 |
| 雷 | 一道折线闪电 | 锐利的折角，起笔突然收笔干脆，转折处有墨点飞溅 |
| 木 | 一片竹叶或嫩芽 | 两笔交叉的叶片，起笔重收笔尖锐 |
| 土 | 一块岩石或山峰 | 沉稳的三角形轮廓，笔触厚重不规则 |
| 混沌 | 一个旋转的太极漩涡 | 连续的螺旋线，墨色从中心向外渐变稀薄 |

### 4.3 天气图标概念

| 天气 | 造型 | 色彩 |
|------|------|------|
| 烈日 | 放射状光线圆 | 橙红 + 金色核心 |
| 灵雨 | 斜线雨滴 + 云朵轮廓 | 青蓝色 |
| 雷暴 | 乌云 + 锯齿闪电 | 灰紫 + 金色电弧 |
| 灵雾 | 层叠的波浪线 | 半透明灰白 |
| 妖风 | 旋转的气流螺旋 | 淡绿 + 风粒子 |
| 寒潮 | 六角雪花结晶 | 冰蓝 + 白色 |
| 蚀月 | 被遮挡的月牙 | 暗紫 + 黑色侵蚀 |
| 七彩祥云 | 彩虹色云团 | 渐变七色 |

## 五、面板与卡片设计语言 (Panel & Card Design)

### 5.1 面板基础模板

**「墨池」面板** — 所有模态面板的基础：

```
┌─────────────────────────────────┐  ← 1px 墨色边框 (ink.deep)
│  ╲                            ╱ │  ← 左上/右上：水墨笔触角饰
│   ╲  ┌───────────────────┐  ╱  │
│    ╲ │     面 标 题       │ ╱   │  ← 标题区：宋体金色 + 下方金线分隔
│     ╲│═══════════════════│╱    │
│      │                   │     │  ← 内容区：bg.panel 底色
│      │   [内容区域]       │     │     垂直滚动，内边距 16px
│      │                   │     │
│     ╱│                   │╲    │
│    ╱ │   [按钮区域]       │ ╲   │  ← 底部按钮区
│   ╱  └───────────────────┘  ╲  │
│  ╱                            ╲ │  ← 左下/右下：水墨笔触角饰
└─────────────────────────────────┘
     ↑ 整个面板带 20px 深色投影
       + 微弱的元素色外发光（根据当前关卡）
```

**面板参数：**
- 背景：`bg.panel` (#0E1018) + 40% opacity
- 边框：`ink.deep` (#1A1C24) 1px 实线
- 圆角：8px（但四角各有水墨笔触装饰覆盖）
- 投影：`0 0 40px rgba(0,0,0,0.8)` + 8px 偏移
- 毛玻璃效果：`backdrop-filter: blur(8px)`（如果引擎支持）

### 5.2 卡片设计语言

**「符箓」卡片** — 用于词条选择、天赋选择、武器祭炼：

```
┌──────────────────────┐
│ ▓▓▓ 品质色顶部条 2px  │  ← 顶部 2px 品质色横线
│                      │
│    🔥 [元素图标]      │  ← 48px 元素图标，带光晕
│                      │
│   ─── 金线分隔 ───   │  ← 1px 金色渐变线 (opacity 50%)
│                      │
│  【词条名称】         │  ← 宋体 Bold，品质色
│   凡品 · 法术         │  ← 小字标签，ink.light 色
│                      │
│  描述文字两行以内...  │  ← 正文，text.primary
│                      │
│  ┌─ 连携提示 ─┐     │  ← 连携/组合提示，半透明底
│  │ 与烈焰掌连携 │     │     元素色文字
│  └────────────┘     │
│                      │
│  ══════════════════  │  ← 底部选择按钮
│    择 此 机 缘       │     宋体 + hover 时粒子涌出
│  ══════════════════  │
└──────────────────────┘
```

**卡片状态：**
- **默认态：** `bg.panel` 底色 + `ink.deep` 边框
- **悬停态：** 边框变为品质色 (opacity 60%) + 4px 品质色外发光 + 卡片微上移 2px
- **选中态：** 品质色边框全亮 + 8px 外发光 + 粒子从卡片边缘涌出
- **禁用态：** opacity 50% + 灰度滤镜

### 5.3 按钮设计

**「印章」按钮** — 主要操作按钮：

```
┌══════════════════════════┐
║    ▶ 踏 入 轮 回        ║  ← 宋体居中，字间距加大
└══════════════════════════┘
  ↑ 金线双线边框 (2px)
    hover 时：边框亮度 +30% + 粒子从边框涌出
    pressed 时：边框缩为 1px + 短暂墨滴爆发粒子
```

**按钮类型：**
- **主按钮 (Primary)：** 金线双框 + `text.gold` 文字 + hover 金色粒子
- **次按钮 (Secondary)：** 墨色单框 + `text.secondary` 文字 + hover 墨色粒子
- **危险按钮 (Danger)：** 血红单框 + `text.blood` 文字 + hover 红色粒子
- **幽灵按钮 (Ghost)：** 无边框 + `text.muted` 文字 + hover 显示墨色底

## 八、覆盖层设计 (Overlay Screens)

### 8.1 天机一击覆盖层 (Crit Moment)

**触发：** 暴击 / 完美闪避 / 连击里程碑 / 万法归一时

**视觉流程：**
1. **冻结帧 (0.05s)：** 画面暂停，轻微去饱和（-20%）
2. **墨痕扩散 (0.1s)：** 从命中点向外扩散一圈水墨晕染
3. **文字显现 (0.1s)：** "天机！" / "完美闪避！" / "万法归一！" 用书法字体从中心缩放弹出
4. **粒子爆发 (0.3s)：** 元素色粒子从文字向四周喷射
5. **恢复 (0.2s)：** 画面恢复，粒子渐隐

**文字样式：**
- "天机！" — 金色书法体 48px + 2px 金色描边 + 4px 墨色投影
- "万法归一！" — 虹彩色 56px + 旋转缩放 + 8px 光晕
- "道之极致！" — 红金色 60px + 全屏脉冲 + 粒子风暴

### 8.2 道消覆盖层 (Death Moment)

**四阶段演出：**
1. **时间凝固 (0.5s)：** 画面去饱和 → 全灰，墨色从四角向中心侵蚀
2. **遗憾标注 (1s)：** 屏幕中央显示"本局遗憾" + 具体遗憾文字，水墨笔触逐字写出
3. **魂魄离身 (1.5s)：** 角色位置出现灵魂上升动画（半透明白色轮廓 + 粒子上升），同心圆图腾扩散
4. **遗言留世 (1s)：** 底部显示告别语 "来世再证大道"，墨迹从两侧合拢

### 8.3 道统觉醒覆盖层 (Dao Tradition Awakening)

**最华丽的仪式：**
1. **全屏暂停 (0.5s)**
2. **金色书法文字逐笔写出：** "焚天真人成道" — 每一笔都拖着金色粒子尾迹
3. **元素色冲击波：** 从角色位置向外扩散的圆环，颜色 = 道统元素色
4. **角色视觉变化：** 角色周围出现持续的元素粒子环绕
5. **墨印落款 (0.3s)：** 右下角出现红色印章 "道统已成"

## 十、粒子特效处方 (Particle Effect Prescriptions)

### 10.1 UI 交互粒子

| 触发 | 粒子效果 | 参数 |
|------|---------|------|
| **按钮点击** | 墨滴爆发 | 5-8 粒子，元素色，从按钮中心向外扩散 30px，寿命 0.4s |
| **卡片悬停** | 底部粒子上升 | 3-5 粒子/s，品质色，从卡片底边上升 20px，寿命 0.6s |
| **卡片选中** | 中心汇聚→爆发 | 12 粒子从四周汇聚 → 停顿 0.1s → 24 粒子向外爆发 60px |
| **品质揭示** | 品质粒子瀑布 | 凡品：无 / 灵品：8蓝色 / 仙品：16紫色 / 天品：24金色 / 道品：32红金色+旋转 |
| **重随** | 金色漩涡 | 16 金色粒子顺时针旋转 360° → 消散 → 新卡片淡入 |
| **跳过** | 墨迹消散 | 卡片向两侧墨迹化消散 → 金色灵石 +30 飘出 |

### 10.2 战斗里程碑粒子

| 触发 | 粒子效果 | 参数 |
|------|---------|------|
| **连击 10** | 小墨溅 | 8 墨色粒子，从连击数字向外扩散 20px |
| **连击 30** | 中墨爆 + 闪白 | 16 元素色粒子 + 0.05s 全屏白闪 |
| **连击 50** | 大墨爆 + 慢动作 | 32 元素色粒子 + 0.2s 慢动作 + 摄像机微震 |
| **连击 100** | 墨风暴 + 缩放 | 64+ 元素色粒子风暴 + 摄像机缩放到 110% → 回弹 |
| **暴击 (天机)** | 时间冻结 + 白闪 + 冲击环 | 0.05s 冻结 → 白色径向冲击环 → 12 粒子向外 |
| **Boss 击杀** | 元素色粒子雨 | 200+ 粒子从天而降 2s，颜色 = Boss 元素色 |
| **道统觉醒** | 全屏墨浪 + 书法 | 100+ 金色粒子 + 书法文字逐笔写出 + 红色印章 |
| **突破** | 金色墨柱冲天 | 从角色位置向上喷射 50 金色粒子柱 + 地面墨圈扩散 |
| **完美闪避** | 时钟涟漪 | 以角色为中心的白色同心圆涟漪 3 层 + 8 粒子 |

### 10.3 粒子渲染规则

- **混合模式：** Additive（加法混合），使粒子在深色背景上发光
- **粒子形状：** 圆形（标准）/ 墨滴形（水墨特化）/ 线段（电弧/风）
- **粒子大小：** 2-8px（UI 交互）/ 4-16px（战斗效果）
- **颜色衰减：** 粒子生命后半段饱和度降低 + 向黑色衰减（模拟墨迹干涸）
- **拖尾：** 高速粒子带 3-5 帧拖尾，拖尾透明度递减

### 13.4 验证方法

1. **启动游戏：** `godot --path game` 从主菜单开始完整流程
2. **视觉一致性检查：** 每个面板是否符合水墨暗色风格
3. **可读性测试：** 战斗中 0.5 秒内能否读懂 HP/冷却/连击
4. **粒子效果检查：** 所有交互粒子是否正常触发
5. **关卡过渡检查：** 5 个关卡的色彩变化是否正确
6. **天象影响检查：** 8 种天象的 UI 效果是否生效
7. **性能检查：** 粒子数量是否导致帧率下降
