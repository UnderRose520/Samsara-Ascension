## 1. 游戏概述

### 1.1 一句话描述

> 一款修仙题材的混合型 Roguelite 游戏，玩家扮演轮回转世的修士，  
> 每一局经历从炼气到飞升的完整修仙人生，通过功法搭配、天象利用、  
> 因果抉择、灵宠并肩、器灵觉醒，打造独一无二的修仙 Build，  
> 最终渡劫飞升——或在无尽深渊中挑战极限。

### 1.2 核心体验

```
┌─────────────────────────────────────────────────────────┐
│                                                         │
│  动作层 —— 操控施法、闪避腾挪、法术连招（手感爽）        │
│  构筑层 —— 功法搭配、词条联动、Build 成型（策略爽）      │
│  表现层 —— 剑光满屏、雷劫降世、万法归一（视觉爽）        │
│  伙伴层 —— 灵宠并肩、器灵觉醒、心意相通（陪伴感）        │
│                                                         │
│  四层体验在同一个战斗中同时发生                          │
│  不是四个系统拼接，而是一个完整体验                      │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

### 1.3 目标受众

- **主要受众：** 喜欢肉鸽游戏的核心玩家（Hades / 杀戮尖塔 / 以撒的结合玩家）
- **次要受众：** 修仙/仙侠文化爱好者
- **潜在受众：** 喜欢 Build 构筑和数值爆炸的休闲玩家
- **长期受众：** 无尽模式排行榜硬核玩家

### 1.4 竞品分析

| 竞品 | 我们学习的 | 我们不同的 |
|------|-----------|-----------|
| Hades | 打击感、叙事与肉鸽结合 | 天象系统、因果系统、无尽模式 |
| 杀戮尖塔 | 词条联动深度、路径选择 | 动作而非卡牌，灵宠/器灵系统 |
| 以撒的结合 | 道具池广度、组合爆炸 | 修仙主题、功法领悟系统 |
| 暖雪 | 修仙美术风格 | 天象系统、因果系统、无尽模式 |
| 吸血鬼幸存者 | 弹幕密度、数值膨胀感 | 主动操作和策略深度 |
| 无尽模式参考：暗黑地牢/雨中冒险 | 无尽循环、排行榜 | 修仙叙事包装、天象动态变化 |

## 2. 核心理念与设计哲学

### 2.1 四个心理钩子

```
钩子 1：可能性的诱惑
  "这局说不定能抽到那个逆天功法……"
  → 随机奖励、隐藏内容、小概率事件、灵宠/器灵

钩子 2：差一点的遗憾
  "就差一个词条就凑成完美 combo 了！"
  → 给玩家看到"接近完成"的构筑，让他不甘心

钩子 3：下次会更好的掌控感
  "我知道该怎么选了，再来一局肯定更强"
  → 局外成长 + 玩家对系统的理解加深

钩子 4：无尽的攀登
  "我还能更强……深渊第 50 层见"
  → 无尽模式的排行榜和极限挑战
```

### 2.2 五项设计原则

```
1. 选择大于运气
   → 所有随机都给玩家"选择权"（三选一而非纯掷骰）
   → 运气决定"你有什么选项"，选择决定"你走什么路"

2. 发现大于告知
   → 词条联动不直接告诉玩家，让玩家自己发现 combo
   → 首次发现隐藏交互时弹出"领悟"提示（惊喜感）

3. 节奏大于强度
   → 不能一直爽，要有弱→强→碾压的节奏
   → 也不能一直弱，每个阶段都要有"进展感"

4. 死亡也是收获
   → 每次死亡都有轮回点数、图鉴更新、解锁进度
   → 死亡界面要让玩家觉得"值了"而非"亏了"

5. 修仙即肉鸽
   → 轮回 = 重开局 / 境界 = 关卡进度 / 功法 = 词条
   → 渡劫 = Boss 战 / 飞升 = 通关 / 深渊 = 无尽模式
   → 所有机制都有修仙叙事包装
```

### 2.3 核心循环

```
一局循环（25-40分钟·主线 / 无上限·无尽）：

  选择灵根/角色
      ↓
  进入修炼（战斗房间）
      ↓
  ┌─→ 击败敌人 → 获得功法/词条选择 → 灵宠/器灵成长 ─┐
  │                                                    ↓
  │              进入下一个房间（战斗/事件/商店/奖励关）
  │                                                    ↓
  │              Boss 战 → 境界突破 ─────────────────┤
  │                                                    ↓
  └────────────────────────────────────────────────────┘
      ↓
  主线：渡劫飞升 → 结算 → 轮回点数 → 局外升级 → 再来一局
  无尽：深渊层 → 结算 → 排名 → 轮回点数 → 再来一局
```

│  打开人物面板     │  C             │  查看词条/属性  │
│  灵宠指令         │  V             │  攻击/跟随/防御│
│  暂停             │  Esc           │  暂停菜单      │
└──────────────────────────────────────────────────┘
```

### 3.2 移动系统

```
基础移动速度：  300 像素/秒
加速状态：      400 像素/秒（持续1秒，触发条件待定）
减速状态：      200 像素/秒（受控效果时）

惯性系统：
  · 松开按键后有 0.05 秒的微滑行（手感更流畅）
  · 不做过度惯性（避免操作延迟感）

碰撞：
  · 角色碰撞体积为圆形，半径 12 像素
  · 与墙壁碰撞后停止（不反弹）
  · 与敌人碰撞时产生轻微推挤力
```

### 3.3 闪避系统（御风步）

```
┌──────────────────────────────────────────────────┐
│  闪避参数                                        │
│                                                  │
│  冲刺距离：      150 像素                         │
│  冲刺时间：      0.2 秒                          │
│  无敌时间：      0.25 秒（覆盖冲刺全程 + 余量）   │
│  冷却时间：      1.0 秒                          │
│  冲刺方向：      当前移动方向 / 鼠标方向（可选）   │
│                                                  │
│  视觉效果：                                       │
│  · 冲刺时角色变为半透明残影                       │
│  · 留下一道灵气轨迹（颜色对应灵根属性）           │
│  · 无敌帧期间角色闪烁                            │
│                                                  │
│  可被词条修改：                                    │
│  · 冷却时间可缩短                                 │
│  · 距离可增加                                     │
│  · 可附加伤害/效果（如雷系闪避放电）              │
│  · 可增加闪避次数（多段闪避）                     │
└──────────────────────────────────────────────────┘
```

### 3.4 灵宠指令系统

```
灵宠指令通过 V 键循环切换：

  攻击模式（默认）：
    · 灵宠自主攻击最近敌人
    · AI：优先攻击你正在攻击的目标

  跟随模式：
    · 灵宠跟在你身后 50 像寸
    · 不主动攻击，但被攻击时反击
    · 适合需要保护灵宠的危险场景

  防御模式：
    · 灵宠环绕你巡逻（半径 80 像素）
    · 优先攻击靠近你的敌人
    · 适合防御型打法
```

### 6.2 伤害计算流水线

```
伤害计算分为 9 个阶段，按顺序执行：

  阶段 1：基础伤害
    base_damage = 攻击力 × 技能倍率

  阶段 2：属性加成
    base_damage ×= (1 + 对应属性加成%)
    // 灵根 +15%，天象 +30%，天赋 +20%，器灵 +10% 等

  阶段 3：词条加成（加法叠加）
    base_damage += 所有"伤害 +X"类词条的总和

  阶段 4：词条乘算
    base_damage ×= 所有"伤害 ×倍率"类词条的乘积

  阶段 5：灵宠/器灵加成
    base_damage ×= (1 + 灵宠被动加成%)
    base_damage ×= (1 + 器灵被动加成%)

  阶段 6：暴击判定
    if random() < 暴击率:
      base_damage ×= 暴击倍率（默认 1.5）
      标记为暴击

  阶段 7：防御减免
    final_damage = base_damage × (100 / (100 + 敌人防御))
    // 穿甲效果在此阶段降低敌人防御

  阶段 8：伤害修正
    final_damage ×= 境界差距修正
    final_damage ×= Boss 特殊修正
    final_damage ×= 无尽模式层数修正（每层 +2% 敌人减伤）

  阶段 9：最终取整 + 后处理
    final_damage = max(1, floor(final_damage))
    → 触发"命中时"类词条/灵宠/器灵效果
    → 触发"暴击时"类词条
    → 触发"击杀"类词条（如果目标死亡）
    → 更新连击计数器
    → 显示伤害数字
```

## 10. 灵宠系统

### 10.1 灵宠概览

```
灵宠 = 跟随战斗的伙伴，有独立生命、技能、成长

  · 灵宠改变战斗方式，不只是额外输出

  · 灵宠可死亡，有保护机制
  · 与词条/天象/器灵深度联动
  · 同时只能出战 1 只，最多收藏 3 只

```

## 11. 器灵系统

### 11.1 器灵概览

```
器灵 = 法宝中沉睡的灵魂，觉醒后法宝能力质变

核心设计：
  · 每个法宝有"沉睡器灵"，特定条件觉醒
  · 觉醒后法宝从被动变为主动技能+人格
  · 器灵有自己的性格和战斗建议
  · 可成长，解锁更多能力
  · 一局最多 1 个器灵觉醒（稀缺性+仪式感）
```

### 11.2 觉醒条件

```
五种共鸣类型：
  1. 战斗共鸣：使用法宝效果累计 N 次
  2. 天象共鸣：特定天象下战斗 N 层
  3. 属性共鸣：某属性达到阈值
  4. 事件共鸣：特殊事件中触发
  5. 因果共鸣：因果标记达到条件

觉醒过程：
  · 满足条件 → 法宝发光 → 屏幕提示 → 按 F 注入灵力
  · 错过需重新积累（进度保留）
  · 觉醒动画 3 秒 + 音效
```

### 11.3 器灵详细设计

```
━━━ 剑魂（玄铁飞剑觉醒）━━━

## 17. 无尽模式

### 17.1 设计理念

```
无尽模式不是主线的"延长"，而是一个独立的游戏模式：

  主线：有终点的修仙人生 → 飞升成功/失败
  无尽：无终点的深渊征途 → 挑战极限，追逐排名

核心问题："飞升之后，还有什么？"
回答："深渊之下，道无穷尽。"

无尽模式满足：
  · 已通关玩家的长期目标
  · Build 极限测试的舞台
  · 排行榜竞争
  · 新词条/灵宠/器灵的获取渠道
```

### 17.2 开启条件与入口

```
开启条件：至少飞升 1 次（完成主线通关）

入口：主界面 → "深渊征途" 按钮

  ┌──────────────────────────────────────────────────┐
  │                                                  │
  │              深 渊 征 途                          │
  │                                                  │
  │  "飞升之后，你发现仙界之下还有无尽深渊。           │
  │   那里封印着远古的恐惧，也隐藏着无穷的力量。"     │
  │                                                  │
  │  选择灵根/角色 → 进入                             │
  │                                                  │
  │  历史最高层数：第 37 层                            │
  │  排名：全服 #128                                  │
  │                                                  │
  └──────────────────────────────────────────────────┘
```

### 17.3 无尽模式结构

```
主线的"境界"系统在无尽模式中变为"深渊层数"：

  ┌─────────────────────────────────────────────────────────┐
  │                                                         │
  │  第 1-10 层   ：深渊浅层    （≈主线炼气期难度）         │
  │  第 11-20 层  ：深渊中层    （≈主线筑基期难度）         │
  │  第 21-30 层  ：深渊深层    （≈主线金丹期难度）         │
  │  第 31-40 层  ：深渊极层    （≈主线元婴期难度）         │
  │  第 41-50 层  ：深渊核心    （超越主线难度）             │
  │  第 51-100 层：混沌深渊    （数值开始指数增长）          │
  │  第 100+ 层  ：无尽虚空    （纯极限挑战）                │
  │                                                         │
  └─────────────────────────────────────────────────────────┘

每层结构：
  · 每层 = 3-5 个房间（比主线更紧凑）
  · 每 5 层 = 1 个小 Boss
  · 每 10 层 = 1 个大 Boss（深渊守卫）
  · 每 25 层 = 1 个域外天魔（无尽专属Boss）
  · 每 50 层 = 天道化身（终极挑战）
```

### 17.4 无尽模式数值缩放

```
核心公式：

  敌人生命 = 基础 × (1.15 ^ 层数)
  敌人攻击 = 基础 × (1.10 ^ 层数)
  敌人防御 = 基础 × (1.05 ^ 层数)

  每层递增：
    敌人生命 +15%
    敌人攻击 +10%
    敌人防御 +5%
    敌人数量 +1%（每层微增）

  实际效果：
    第1层：敌人生命 100
    第10层：敌人生命 ~404
    第25层：敌人生命 ~3,292
    第50层：敌人生命 ~108,366
    第100层：敌人生命 ~1,173,908,529（近12亿）

  这意味着：
    · 前20层：和主线体验类似，构筑成型
    · 21-40层：需要完善的 Build 才能推进
    · 41-60层：需要完美的 Build + 高级灵宠/器灵
    · 61-100层：需要极限操作 + 极致 combo
    · 100+层：纯数值压力，需要特殊策略和运气

玩家属性缩放：
  · 词条无上限（无尽模式词条槽固定12个，但词条效果不封顶）
  · 灵宠/器灵持续成长（无尽模式中成长速度加快）
  · 局外永久成长在无尽模式中仍然生效
  · 无尽模式专属词条/天赋出现
```

### 17.5 无尽模式专属系统

```
━━━ 无尽模式专属天赋（飞升后解锁）━━━

每局开始时生成，暗中影响词条/天气/事件权重。
玩家不直接看到，但逐渐感受到"这局的氛围"。

天命种子列表：
  火德当兴 → 火系+40%，烈日+30%，火系事件+50%
  水泽万物 → 水系+40%，灵雨+30%，回复+20%
  雷霆万钧 → 雷系+40%，雷暴+30%，暴击+5%
  妖孽横行 → 精英+50%，精英掉品质+1，金币+20%
  仙缘深厚 → 稀有事件×2，七彩+50%，商店品质+1
  因果纠缠 → 因果事件×2，高风险奖励+30%，心魔标记-1
  天劫将至 → Boss后额外小天劫挑战
  灵气枯竭 → 灵力恢复-50%，灵力上限+80%，丹药+30%
  百族争鸣 → 属性均等，多属性词条+50%
  混沌天命 → 5%概率，每层随机应用一个其他天命
  深渊意志 → 无尽模式专属，每5层随机切换天命效果
```

## 20. 动态难度系统

```
系统实时评估"构筑完成度"（词条数量/品质/功法/联动/属性）

  构筑很强（>80%）→ 敌人+10%血+5%攻，精英+15%，Boss多1技能
  构筑偏弱（<40%）→ 敌人-10%血-5%攻，商店更友好，Boss血-10%

调节幅度±10-15%，玩家不感知，不影响词条品质和成就。

无尽模式中动态难度更敏感：
  · 每5层重新评估
  · 如果玩家过强（连续3层无伤），下一层环境效果更强
  · 如果玩家过弱（连续3层死亡复活），下一层略微削弱
  · 保持"刚好在能力边缘"的紧张感
```
