## 1. 游戏概述

### 1.1 一句话描述

> 一款修仙题材的混合型 Roguelite 游戏，玩家扮演轮回转世的修士，每一局经历从炼气到飞升的完整修仙人生。通过功法搭配、天象利用、地形交互、因果抉择、灵宠并肩、器灵觉醒，打造独一无二的修仙 Build，最终渡劫飞升——或在无尽深渊中挑战极限。

### 1.2 核心体验

```
┌─────────────────────────────────────────────────────────┐
│                                                         │
│  动作层 —— 操控施法、闪避腾挪、法术连招（手感爽）        │
│  构筑层 —— 功法搭配、词条联动、Build 成型（策略爽）      │
│  表现层 —— 剑光满屏、雷劫降世、万法归一（视觉爽）        │
│  环境层 —— 天象变化、地形交互、借势而行（策略+操作爽）    │
│  伙伴层 —— 灵宠协同、器灵觉醒、心意相通（陪伴感）        │
│                                                         │
│  五层体验在同一个战斗中同时发生                          │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

### 1.3 核心卖点（差异化标签）

```
宣传核心标签：

  "天地为棋，万法归一"

  · 天象不是背景，是你的武器
    → 积水导雷、干燥引爆、冰面滑行、毒雾融合
    → 每一局的天象都在改变你的战斗方式

  · 词条不是堆叠，是化学反应
    → 灼烧 × 引燃 × 爆燃 × 火借风势 = 一发子弹烧全场
    → 50+ 种联动组合，玩家自己发现

  · 灵宠不是摆设，是战友
    → V+方向键触发协同技能
    → 火灵雀冲锋留火带、雷灵貂瞬移雷击、土灵龟为你挡刀

  · 死亡不是结束，是下一世的开始
    → 前世遗泽引导方向、轮回点数永久成长、因果系统延续

  · 每局都是不同的命运
    → 天命种子暗中塑造你的道路，身世来历赋予独特起点
    → 10种灵根×6个角色×3种道心 = 截然不同的修仙之旅
```

### 1.4 目标受众

- **主要：** 肉鸽核心玩家（Hades / 杀戮尖塔 / 以撒的结合）
- **次要：** 修仙/仙侠文化爱好者
- **潜在：** 喜欢 Build 构筑和数值爆炸的休闲玩家
- **长期：** 无尽模式排行榜硬核玩家

### 1.5 竞品分析

| 竞品 | 学习的 | 不同的 |
|------|--------|--------|
| Hades | 打击感、叙事融合 | 天象地形交互、因果系统 |
| 杀戮尖塔 | 词条联动深度 | 动作而非卡牌，灵宠/器灵 |
| 以撒的结合 | 道具池组合爆炸 | 修仙主题、功法领悟 |
| 暖雪 | 修仙美术风格 | 天象系统、灵宠协同、无尽模式 |
| 吸血鬼幸存者 | 弹幕密度、数值膨胀 | 主动操作和策略深度 |
| 修仙模拟器 | 随机天赋×构筑驱动 | 动作战斗、实时操作、roguelite循环 |

## 2. 核心理念与设计哲学

### 2.1 四个心理钩子

```
钩子 1：可能性的诱惑
  "这局说不定能抽到那个逆天功法……"
  → 随机奖励、隐藏内容、小概率事件、灵宠/器灵

钩子 2：差一点的遗憾
  "就差一个词条就凑成完美 combo 了！"
  → Combo 指示器让玩家看到"完成度 75%"的 build

钩子 3：下次会更好的掌控感
  "我知道该怎么选了，再来一局肯定更强"
  → 前世遗泽 + 局外成长 + 系统理解加深

钩子 4：无尽的攀登
  "我还能更强……深渊第 50 层见"
  → 无尽模式的排行榜和极限挑战
```

### 2.2 七项设计原则

```
1. 选择大于运气
   → 所有随机都给玩家"选择权"
   → 难度让玩家自己选（开局选择 + 心魔试炼）

2. 发现大于告知
   → 词条联动不直接告知，让玩家自己发现 combo
   → 天象 × 地形 × 词条的交互等待玩家探索

3. 节奏大于强度
   → 不能一直爽，要有弱→强→碾压的节奏
   → Boss 门前有"呼吸点"，战斗内有高潮低潮

4. 死亡也是收获
   → 前世遗泽引导下一局方向
   → 轮回点数永久成长
   → 名场面回顾让死亡有记忆点

5. 修仙即肉鸽
   → 轮回/境界/功法/渡劫/飞升 全部对应肉鸽机制

6. 每一秒都有意义
   → 战斗结束直接选路径（无跑图）
   → 过渡动画同时传递信息
   → Boss 门前暂停是"有意义的调息"

7. 透明的掌控感
   → 难度玩家自己选（开局 + 心魔试炼）
   → Build 可视化四层体系
   → 灵宠/器灵成长全程可见
   → 因果标记 HUD 实时可见
```

### 2.3 核心循环

```
一局循环（15-30分钟·主线 / 无上限·无尽）：

  选择难度 → 选择灵根/角色 → 身世天命 → 前世遗泽（如非首局）
      ↓
  进入修炼（战斗房间）
      ↓
  ┌─→ 击败敌人 → 直接选择下一路径 → 灵宠/器灵成长 ─┐
  │                                                  ↓
  │          进入下一个房间（战斗/事件/商店/奖励关）
  │                                                  ↓
  │          Boss 门前调息 → Boss 战 → 境界突破 ────┤
  │                                                  ↓
  └──────────────────────────────────────────────────┘
      ↓
  主线：渡劫飞升 → 结算 → 轮回点数 → 局外升级
  无尽：深渊层 → 结算 → 排名 → 轮回点数
      ↓
  再来一局
```

## 3. 难度系统

### 3.1 设计理念

```
难度系统的核心矛盾：
  · 新手需要低门槛进入 → 不能一上来就难
  · 老手需要挑战感     → 不能一直简单
  · 玩家讨厌"被看低"   → 不能用"简单/普通/困难"这种标签
  · 玩家喜欢掌控感     → 让玩家自己选，而非系统暗改

解决方案：用修仙叙事包装难度选择

  不是"选难度"
  而是"选择你的道心"
```

### 3.4 难度选择的心理设计

```
关键原则：不要让玩家觉得"选简单 = 丢人"

  1. 没有"简单/普通/困难"这种直白标签
     → 用修仙术语包装："问道/悟道/证道"
     → 每个都有正面描述，没有贬义

  2. 每个难度都有独特价值
     → 问道：适合体验剧情和学习系统
     → 悟道：标准修仙体验
     → 证道：有额外奖励和称号
     → 不是"弱者选简单"，而是"不同道心不同路"

  3. 可以随时修改
     → 设置里随时切换（下一层生效）
     → 不会被"锁定"在某个难度
     → 鼓励玩家"先问道入门，再证道挑战"

  4. 无尽模式难度独立
     → 无尽模式不使用道心系统
     → 而是用层数自然递增难度
     → 硬核玩家的挑战在无尽模式中

  5. 首次游戏推荐问道
     · 第一次进入游戏时，默认选中"问道"
     · 下方提示："推荐首次游玩选择此道心"
     · 但不强制，玩家可以自由选择
     · 选择证道时额外提示："此道心适合有经验的修士"

  6. 道心选择后可查看完整影响
     · 选中某个道心后，展开"详细影响"面板
     · 列出所有数值变化
     · 信息完全透明
```

### 3.5 心魔试炼（游戏内难度调节）

```
心魔试炼不是每关必出，而是事件型可选挑战：

  触发条件：
    · 问道难度：不触发（新手保护）
    · 悟道难度：每关 20% 概率出现在事件房
    · 证道难度：每关必出 1 次

  ┌──────────────────────────────────────────────┐
  │                                              │
  │  房间中央矗立着一面古老的铜镜。              │
  │  镜中映出你的身影，但眼神不同……              │
  │                                              │
  │  ┌──────────────────────────────────────┐   │
  │  │                                      │   │
  │  │  选项 A：接受心魔试炼                 │   │
  │  │  → 本层剩余房间敌人全属性 +15%        │   │
  │  │  → 但本层所有词条品质 +1 阶           │   │
  │  │  → 击败 Boss 后获得心魔碎片           │   │
  │  │  → 获得因果标记"逆天"                 │   │
  │  │                                      │   │
  │  │  选项 B：参悟铜镜                     │   │
  │  │  → 消耗 30 灵石                      │   │
  │  │  → 获得 1 个随机词条（正常品质）      │   │
  │  │                                      │   │
  │  │  选项 C：离开                         │   │
  │  │  → 什么都不发生                       │   │
  │  │                                      │   │
  │  └──────────────────────────────────────┘   │
  │                                              │
  └──────────────────────────────────────────────┘

  心魔碎片用途：
    · 收集 3 个 → 下一局可选择"心魔强化"开局
    · 心魔强化：开局获得 1 个仙品词条，但敌人 +10%
    · 是追求极限的玩家的额外挑战+奖励通道
```

### 3.6 无尽模式难度（自然递增 + 显式选择）

```
无尽模式不使用道心系统，用自然递增 + 显式三档选择：

  自然递增（基础）：
    敌人每层 +15%生命 +10%攻击 +5%防御

  每 10 层前可选择：
    ┌──────────────────────────────────────────────┐
    │                                              │
    │  你来到了深渊的第 20 层……                     │
    │  前方的灵气波动更加强烈。                     │
    │                                              │
    │  [挑战模式] 敌人+20%血，每层额外掉1词条      │
    │  [标准模式] 正常难度                         │
    │  [沉稳模式] 敌人-10%血，排名不计入主榜       │
    │                                              │
    └──────────────────────────────────────────────┘

  · 挑战模式：追求更高排名的玩家
  · 标准模式：正常体验
  · 沉稳模式：练习/休闲，但排名有代价（不计入主榜）
```

### 12.2 灵宠获取

```

  · 你血量<20%：V+S 冷却减半，效果更强

## 13. 器灵系统

### 13.1 器灵概览

```
器灵 = 法宝中沉睡的灵魂
  · 特定条件"共鸣"觉醒
  · 觉醒后法宝能力质变：被动→主动技能+人格
  · 有性格和战斗建议（可选关闭）
  · 可成长
  · 一局最多1个器灵觉醒
```

### 13.2 觉醒条件

```
五种共鸣：
  1. 战斗共鸣：使用法宝效果累计N次
  2. 天象共鸣：特定天象战斗N层
  3. 属性共鸣：某属性达阈值
  4. 事件共鸣：特殊事件触发
  5. 因果共鸣：因果标记达条件

觉醒：满足条件→法宝发光→提示→按F注入灵力→3秒动画

隐性保底：
  · 前4关中，至少有1种共鸣条件可被自然触发（如匹配天象、对应事件出现等）
  · 不强制给器灵，但确保机会存在
```

### 13.3 器灵列表

```
剑魂（玄铁飞剑）：飞剑命中200次 → 万剑归宗剑阵
  人格：沉默冷静 / 成长：伤害+→剑意领域→驻留飞剑

雷灵（天雷珠）：雷暴天30次落雷 → 引雷入体指定落点
  人格：高傲自称"本座" / 成长：雷伤+→静电场→九天雷劫

## 19. 无尽模式

### 19.1 开启条件

```
至少飞升1次 → 主界面解锁"深渊征途"
```

### 19.2 结构

```
每层 3-5 房间（比主线紧凑）
每5层小Boss / 每10层大Boss / 每25层域外天魔 / 每50层天道化身

层数区域：
  1-10 浅层 / 11-20 中层 / 21-30 深层 / 31-40 极层
  41-50 核心 / 51-100 混沌 / 100+ 无尽虚空
```

### 19.3 数值缩放

```
敌人生命 = base × 1.15^层 / 攻击 = base × 1.10^层 / 防御 = base × 1.05^层

第10层生命~404 / 第25层~3292 / 第50层~108366 / 第100层~11.7亿

高层机制防御（防止纯数值碾压）：
  · 21层起：精英获得"韧性"（单次伤害不超过最大生命15%）
  · 31层起：敌人获得"灵气护体"（每5秒生成最大生命3%护盾）
  · 41层起：Boss获得"道心不灭"（生命低于10%时2秒无敌，每场触发1次）
  · 51层起：所有敌人+1%全抗性/层（叠加，可被"道之极致"词条无视）

这些机制不设玩家伤害上限，而是通过敌方机制层防御让高层战斗
既有策略深度，又保留build数值爆炸的成就感。
```

### 19.4 无尽专属内容

```
5个专属天赋（无极之道/不死不灭/天地归墟/深渊之眼/轮回不灭）
6个专属词条（深渊共鸣/虚空之力/无尽回响/永恒之火/时间裂隙/道之极致）
2个专属灵宠（深渊灵蝶/虚空灵狐）
2个专属事件（深渊意志/轮回商人）
每10层难度三档选择（挑战/标准/沉稳）
混沌天象（51+层多重天象叠加/逆天象）
```

### 19.5 排行榜

```
排名方式：最高层数 / 最速通关 / 最低装备 / 特定角色
每月赛季：轮回点数+限定称号+灵宠皮肤
```

### 19.6 每日挑战

```
每日挑战模式：
  · 每日0点刷新一个固定种子挑战（所有玩家相同）
  · 10层紧凑流程，天命种子/身世/天象全部固定
  · 排名方式：通关时间 / 最高连击 / 最少受伤
  · 奖励：轮回点数（前10%额外×2）+赛季积分
  · 入口在主界面"深渊征途"下方，独立于无尽模式
  · 可回看昨日挑战（无排名但可练习）
```

## 23. 动态难度系统

### 23.1 设计理念

```
三层难度体系：

  第一层：道心选择（开局，玩家主动选）
  第二层：心魔试炼（游戏内，事件型可选挑战）
  第三层：隐性调节（后台微调，玩家不感知）

三层叠加，让难度"刚好在你能力边缘"。
```

### 23.2 隐性调节（后台）

```
实时评估"构筑完成度"（词条/品质/功法/联动/属性加权）

  构筑很强(>80%)：敌人+5%生命+3%攻击，精英+10%，Boss多1技能
  构筑偏弱(<40%)：敌人-5%生命-3%攻击，商店更友好，Boss血-5%

关键规则：
  · 调节幅度极小（±5%），玩家不感知
  · 只调敌人生命，不调掉落品质（完全公平）
  · 不影响成就和解锁
  · 问道难度调节范围更大（±10%）
  · 证道难度调节范围更小（±3%）
```

## 28. 数据架构

### 28.1 三层架构

```
配表层（策划用）：CSV/Excel，每行一条效果，最简单
  id        name    type   element  quality  trigger    effect      value
  fire_ball 火焰弹  spell  fire     common   on_hit     apply_status burn
  fire_ball 火焰弹  spell  fire     common   on_attack  add_damage   15

枚举层（程序用）：
  enum TriggerType { OnAttack, OnHit, OnKill, OnDodge, Passive, ... }
  enum EffectType { AddDamage, ApplyStatus, ModifyStat, AoeDamage, ... }
  enum Element { Fire, Water, Thunder, Wood, Earth, None, Mixed }
  enum Quality { Common, Rare, Epic, Legendary, Dao }
  enum StatusType { Burn, Freeze, Paralyze, Poison, Slow, ... }

运行时层（引擎用）：
  struct CompiledTag { id, trigger, effect, element, value, ... }
  配表时编译一次，运行时直接用结构体
```
