# 《轮回仙途》玩法深挖与爽点重设计案

> 目标：把现有“修仙 + Roguelite + 涌潮战斗”的系统堆叠，收束成一套玩家每 5 秒、30 秒、3 分钟、15 分钟都能感到爽和有目标的玩法机器。

## 1. 当前诊断

现有设计已经有足够大的世界观和系统储备：道心难度、五重境界、功法词条、天象地形、灵宠、器灵、因果、道统、前世遗泽、无尽和秘境模式。当前 Godot 原型也已经实现了战斗房涌潮、Q/E/R 法术、词条三选一、Combo、道统、天象地形、灵宠、事件、五关流程和轮回成长。

真正的问题不是“系统不够多”，而是爽点还没有被组织成稳定节奏：

- 战斗爽点偏散：杀怪、法术、天象、词条各自有效，但缺少统一的高潮触发器。
- 构筑爽点偏后置：道统很有吸引力，但玩家早期不一定知道自己快成型了。
- 天象爽点偏数值：当前配表已有雨/雷/火/风等加成，但玩家需要更强的“借天地杀敌”的感知。
- 涌潮压力偏纯粹：杀够数量或拖到时间可过关，但每个战斗房的叙事和节奏差异还不够。
- 系统体量偏大：角色、器灵、丹药、秘境、PvP 都很诱人，但主线核心还没完全打透前，不宜继续横向扩张。

## 2. 新核心定位

### 一句话体验

玩家不是在一间间房里清怪，而是在“借天地之势、炼自己的道”。

每一局的爽感由四个层级构成：

1. **瞬时爽：** 命中、暴击、穿透、一串敌人被清掉。
2. **战术爽：** 把敌人引入天象地形，用法术点燃、导雷、冻结、爆开。
3. **构筑爽：** 词条从散件变成套路，套路变成道统，角色从凡修变成一派宗师。
4. **轮回爽：** 死亡不亏，下一世带着明确方向再开。

建议新的宣传内核改成：

> “不是捡词条变强，而是把整片天地炼成你的功法。”

## 3. 爽点节奏引擎

### 5 秒循环：手感反馈

玩家每 5 秒至少应看到一次明显正反馈：

- 普攻穿透一排敌人。
- 法术击中触发属性色爆点。
- 灵宠自动补刀。
- 连击数字跳档。
- 地面小范围爆燃、导电、冰裂。

落地规则：

- 每个敌人死亡时必须有“灵气碎裂”反馈，而不是简单消失。
- 10/30/60/100 连击给不同 UI 字号和音效，不必都慢动作。
- 敌人被状态击杀时显示死法标签：`焚灭`、`雷殛`、`冻裂`、`毒蚀`、`斩落`。

### 30 秒循环：涌潮小高潮

每个战斗房不只是持续刷怪，而是三幕式：

1. **探势，0-10 秒：** 低密度敌人，展示当前天象/房间地形。
2. **聚势，10-45 秒：** 敌人加压，出现一个明确战术目标。
3. **破势，最后 15 秒或击杀配额 80%：** 精英/阵眼/天雷/灵泉等高潮事件出现。

战斗房可随机抽一个“房间势”：

| 房间势 | 规则 | 爽点 |
|---|---|---|
| 雷池将满 | 场上积水逐渐扩大，雷系命中会连锁 | 一发雷清半屏 |
| 烈阳裂地 | 干燥区不断出现，火系命中爆燃 | 拉怪进火场后引爆 |
| 风眼游移 | 风眼推动敌我弹道 | 顺风放法术像开炮 |
| 魔潮阵眼 | 场上出现 1-3 个阵眼，打掉后全屏震荡伤害 | 优先级决策和爆屏 |
| 灵泉争夺 | 中央灵泉周期治疗敌我 | 抢位置、回血反杀 |
| 心魔窥伺 | 低血时出现心魔影，击杀给高品质奖励 | 风险换爆发 |

### 3 分钟循环：构筑里程碑

每 2-3 个房间，玩家必须看到自己构筑发生质变：

- 解锁新法术槽或换绑法术。
- 功法升层。
- Combo 进度从 1/3 到 2/3。
- 道统差 1 件时给“道统浮现”提示。
- 灵宠获得阶段成长。

关键不是直接送强，而是让玩家看见“我快成了”。

### 15-30 分钟循环：一世成道

主线一局建议明确为五段爽感：

| 阶段 | 玩家心理 | 设计目标 |
|---|---|---|
| 炼气 | 我先活下来 | 给基础杀怪爽和第一个 Combo 方向 |
| 筑基 | 我知道这局玩什么了 | 天象交互加入，开始借势 |
| 金丹 | 我的 Build 成型了 | 出第一个质变词条/灵宠成长 |
| 元婴 | 我开始碾压 | 涌潮密度提升，给大面积清屏 |
| 渡劫 | 证明这套道能不能成 | Boss/天劫检验，允许极限爆发 |

## 4. 战斗重设计

### 玩家核心资源：道势

新增一个轻量资源 `道势`，用于统一战斗高潮。

获得方式：

- 连击不断：持续积累。
- 利用天象击杀：大量积累。
- 精准闪避敌人重击：大量积累。
- 灵宠协同命中多个敌人：积累。

消耗方式：

- `满道势` 自动进入 5 秒“道法通明”：攻速、法术 CD、特效强度提升。
- 玩家也可主动按一个键释放“万法归一”：把当前最高完成度 Combo 变成一次清屏级爆发。

设计价值：

- 把连击、闪避、天象、灵宠、词条全接到一个爽点出口。
- 新手靠杀怪也能触发，高手通过天象和闪避更快触发。
- 给屏幕爆发一个合理频率，避免一直平。

### 普攻与法术定位

当前 Q/E/R 已经具备基础差异，但还可强化成“普攻铺垫，法术引爆”：

- 普攻负责叠状态、维持连击、补刀。
- Q 是短 CD 主输出。
- E 是穿透/控制/导雷工具。
- R 是大范围/高伤/地形改造工具。

例子：

- 火系：普攻叠灼烧，Q 点燃，E 穿透传播，R 爆燃。
- 雷系：普攻麻痹，Q 单点雷印，E 连环导电，R 天雷落点。
- 冰系：普攻减速，Q 冰刺，E 冻结积水，R 冰裂爆开。

## 5. 词条与道统重设计

### 词条不再只是属性，而是三段成道

建议把构筑表达改成：

1. **道胚：** 1 个核心词条，告诉玩家这局方向。
2. **道脉：** 2-3 个联动词条，让玩法开始变化。
3. **道统：** 4-5 个条件达成，给称号、外观、强被动和终局目标。

UI 上不要只显示 Combo 标签，而显示“道统谱系”：

```text
焚天道统
已得：火焰弹、引燃
缺少：引爆类、扩散类、纯阳类
当前进度：2/5，道火初燃
```

未发现过的具体词条可隐藏，但“类别”要显示。这样既保留探索，又不让玩家迷路。

### 词条选择要形成有趣抉择

三选一建议固定三种心理位置：

- **立刻变强：** 攻击、攻速、生命、穿透等。
- **构筑推进：** 当前道统/Combo 相关。
- **风险奇遇：** 高品质但带代价，或偏离路线但很诱人。

当玩家某个道统进度 ≥ 60%，系统应提高相关词条权重，但仍保留偏离诱惑。

### 首批重点道统

先不要追求 50+ Combo。建议第一阶段只打磨 6 个“能明显改变玩法”的道统：

| 道统 | 核心爽点 | 终局表现 |
|---|---|---|
| 焚天道统 | 火焰传播和爆燃 | 满屏连环爆炸 |
| 雷罚道统 | 导电、弹射、落雷 | 雷池秒清怪群 |
| 沧海道统 | 冻结、护盾、冰裂 | 控场后碎冰爆杀 |
| 万毒道统 | 持续毒云和回血 | 越打越稳，毒雾吞场 |
| 不动道统 | 护盾反伤和站桩 | 被围攻反而爆发 |
| 五行归元 | 多属性轮转 | 触发“万法归一”大爆发 |

## 6. 天象重设计

### 从“天气 Buff”改成“天地机关”

每种天象都要回答三个问题：

1. 玩家怎么看懂？
2. 玩家怎么利用？
3. 利用成功有什么大反馈？

建议设计：

| 天象 | 可读信号 | 玩家动作 | 爽点反馈 |
|---|---|---|---|
| 灵雨 | 蓝色水洼扩张 | 把敌人引进水，雷/冰命中 | 电弧扩散或整片冻结 |
| 烈阳 | 裂地干燥区发红 | 火系点燃地面 | 区域爆燃连锁 |
| 雷暴 | 地面金圈预警 | 站位诱导或雷法牵引 | 天雷劈中怪群 |
| 妖风 | 风线和尘带方向 | 顺风发弹道/击退 | 弹道加速贯穿 |
| 灵雾 | 雾墙遮挡边界 | 近战/毒系进入雾中 | 雾中毒爆、偷袭加成 |
| 霜雪 | 冰面反光 | 滑步闪避、冰系扩散 | 冰裂扇形爆开 |

### 借势按钮

可考虑把天象利用做成轻量主动机制：当玩家站在特殊地形上，短按交互键触发“借势”。

例子：

- 水洼借势：下一次雷/冰法术强化。
- 干燥借势：脚下生成火环。
- 风眼借势：短距离顺风突进。
- 冰面借势：无敌滑步。

这能让玩家从“天气随机影响我”变成“我主动借天地打人”。

## 7. 涌潮与敌人重设计

### 敌人职责

敌人不要只按近战/远程区分，而要形成战斗问题：

| 敌人类型 | 给玩家的问题 | 对应爽点 |
|---|---|---|
| 追猎者 | 逼玩家移动 | 闪避后反杀 |
| 弹幕手 | 控制空间 | 穿过弹幕接近击杀 |
| 护阵者 | 给怪群护盾 | 优先击杀后全体破防 |
| 自爆魔 | 制造爆点 | 引到怪群中炸怪 |
| 召唤者 | 拖时间会失控 | 快速斩首 |
| 精英镜像 | 复制玩家一种能力 | 打自己的 Build 影子 |

当前已有 berserker/ranged/sniper/skirmisher/shaman 等 archetype，可以直接往这些职责上靠。

### 精英词缀要可感知

精英词缀建议做到“一眼懂，一打爽”：

- 火甲：靠近灼烧，但冰系击破后爆开。
- 雷核：周期放电，站在水中更危险，但击杀后全屏导雷。
- 护法：给小怪护盾，死亡时小怪全部眩晕。
- 贪灵：会吸收场地灵气变大，击杀掉更多灵石。

## 8. Boss 重设计

每个 Boss 是一堂课，检验前面关卡学到的东西。

| Boss | 检验主题 | 关键机制 | 击败爽点 |
|---|---|---|---|
| 赤焰守关傀儡 | 基础移动和爆燃 | 冲锋留下干燥裂地 | 引爆裂地破甲 |
| 幽泉秘境镇守 | 积水/冻结/导雷 | 场地水位周期变化 | 冻住水面打断大招 |
| 天雷劫影 | 预警圈和牵雷 | 落雷可被玩家法术牵引 | 反引天雷劈 Boss |
| 焚心魔将 | 高压涌潮和心魔 | 召唤玩家影子 | 击杀影子获得道势 |
| 九天雷劫化身 | 终局综合考试 | 九道劫，每道对应天象 | 第九劫用万法归一破天 |

## 9. 灵宠与器灵收束

灵宠和器灵都很有戏，但当前主线建议先做“少而精”：

- 灵宠负责陪伴与协同，玩家按 V 触发强力动作。
- 器灵负责一次局内觉醒，按 F 释放高仪式感技能。
- 一局只强调 1 只灵宠 + 1 个器灵，避免控制负担。

灵宠协同应接入道势：

- 协同击中 3 个以上敌人：大量道势。
- 协同触发天象：额外道势。
- 协同救命：给短暂无敌和大字反馈。

## 10. 局外与死亡重设计

死亡结算不要只给资源，应给“下一局方向”：

- 本局最强 Combo：可选择一个降阶遗泽。
- 本局最精彩击杀：生成名场面标题。
- 本局差一点完成的道统：下局提高首个相关道胚出现率。

示例：

```text
道消于金丹 · 雷罚未成
本局遗憾：雷罚道统 3/4，缺少「雷体类」
前世遗泽：
1. 残雷入骨：开局获得雷系道胚
2. 未竟雷罚：前 2 关雷系词条权重 +25%
3. 劫后余灵：开局道势上限 +10%
```

这样死亡会变成“下一世我知道怎么追了”。

## 11. 当前原型优先改造路线

### P0：立刻提升爽感

1. 增强死亡反馈：死法标签、属性爆点、灵气碎裂。
2. 增加连击档位：10/30/60/100 连击反馈。
3. 战斗房三幕式：探势、聚势、破势。
4. 每个房间随机一个“房间势”。
5. Combo/道统 UI 显示“缺少类别”，不是只显示 tag。

### P1：把系统串起来

1. 新增道势资源。
2. 天象击杀、闪避、灵宠协同都产出道势。
3. 满道势触发“道法通明”短爆发。
4. 主动释放“万法归一”作为构筑大招。
5. 天象地形加入明确的爆燃/导电/冻结反馈。

### P2：扩内容但不扩复杂度

1. 先打磨 6 个核心道统。
2. 每个道统配 4-6 个强相关词条。
3. 每关 Boss 对应一个天象/战术主题。
4. 灵宠先扩到 3 只：火、雷、水，分别服务三套主流构筑。
5. 器灵先做 2 个：剑魂和雷灵，验证觉醒体验。

### P3：长期扩展

1. 角色与灵根。
2. 更多灵宠/器灵。
3. 上古秘境 PvE，再考虑 PvPvE。
4. 无尽排行榜和每日挑战。

## 12. 设计取舍

建议暂缓：

- PvP/秘境完整模式：很有潜力，但会吞掉大量系统设计和工程成本。
- 过多角色机制：主线战斗还没完全打爽前，角色差异会放大平衡负担。
- 50+ Combo 一次性铺开：玩家难以感知，调试成本也高。
- 复杂丹药/法宝池：先让“词条 + 天象 + 灵宠 + 道势”跑顺。

建议坚持：

- 修仙术语必须服务玩法，不只是换皮。
- 每个新增系统都必须接入至少一个爽点循环。
- 玩家每局都要有一个清晰追求：这局我要成哪条道。
- 天象必须可被主动利用，否则就只是天气 Buff。

## 13. 最小可验证 Demo

如果只做一个两周可验证版本，建议目标是：

> 一局 10 分钟，3 关，3 套道统，3 种天象。玩家能稳定体验到“引怪入地形 -> 法术引爆 -> 道势满 -> 万法归一清屏 -> 道统觉醒”的完整爽点链。

验收标准：

- 新玩家 5 分钟内能说出“我刚才用天气杀了一堆怪”。
- 玩家每个战斗房至少触发 1 次明显高潮。
- 玩家死亡后知道下局想追什么构筑。
- 玩家第一次觉醒道统时愿意截图或复述。

## 14. 深度打磨原则

这一轮打磨的核心不是“再加系统”，而是让所有系统都服务同一条体验链：

```text
看见机会 -> 做出操作 -> 触发连锁 -> 得到爆发 -> 推进构筑 -> 形成记忆
```

每个玩法点都要经过四个问题检验：

1. **我能不能一眼看懂机会？**
2. **我有没有一个明确动作可以利用机会？**
3. **利用成功后是否有足够夸张的反馈？**
4. **这次成功是否推进了本局构筑或下一次选择？**

如果一个设计只能回答“数值变强了”，但不能回答“玩家刚才做了什么漂亮操作”，就先不要做。

### 玩家主行为

主线战斗的玩家动词应固定为六个：

| 动词 | 含义 | 对应爽点 |
|---|---|---|
| 聚 | 走位把怪群拉到一起 | 为爆发创造前置 |
| 借 | 借天象/地形获得强化 | 修仙味最强的策略点 |
| 叠 | 普攻和小法术叠状态 | 看到敌人逐渐进入可引爆状态 |
| 破 | 用大法术/地形触发连锁 | 大片击杀和视觉高潮 |
| 遁 | 闪避、滑步、顺风位移 | 高压下活下来 |
| 成 | 词条、功法、道统质变 | 本局成长成就 |

战斗不应该鼓励玩家原地站桩自动射击。自动瞄准/自动普攻可以降低门槛，但真正的高爽点必须来自“聚怪、借势、引爆、逃生”的主动行为。

## 15. 战斗手感规格

### 基础攻击定位

| 行为 | 设计定位 | 手感要求 |
|---|---|---|
| 普攻 | 维持连击、叠状态、补刀 | 高频、稳定、反馈清楚 |
| Q | 低 CD 主输出 | 命中感重，适合频繁按 |
| E | 直线穿透/导雷/传播 | 有“穿一串”的爽感 |
| R | 大范围控制/引爆/地形变化 | CD 稍长，但每次都该有戏 |
| Space | 保命 + 完美闪避产出 | 不是纯逃跑，闪好能反打 |
| V | 灵宠协同 | 少按但强，像队友出手 |
| 道势爆发 | 局内高潮出口 | 5-8 秒内让玩家明显变成“强者” |

### 单次命中反馈

单次命中至少分三档：

| 档位 | 触发 | 表现 |
|---|---|---|
| 轻击 | 普通命中 | 小飘字 + 轻粒子 |
| 重击 | 暴击、穿透 2 个以上、状态命中 | 大飘字 + 属性色爆点 + 轻震 |
| 破势 | 状态引爆、地形连锁、精英破甲 | 屏幕边缘属性光 + 大字短闪 + 音效抬升 |

死亡反馈必须带“死因”：

| 死因 | 触发条件 | 文案 |
|---|---|---|
| 火 | 灼烧/爆燃击杀 | 焚灭 |
| 雷 | 麻痹/导电/落雷击杀 | 雷殛 |
| 水冰 | 减速/冻结/冰裂击杀 | 冻裂 |
| 毒 | 毒/木系持续伤害击杀 | 毒蚀 |
| 暴击 | 天机一击击杀 | 天机 |
| 道势 | 道法通明/万法归一击杀 | 归一 |

这类标签不只是 UI 装饰，它会告诉玩家：“我刚刚不是普通打死了敌人，而是用这套玩法打死了敌人。”

### 连击重新定位

当前已有连击里程碑，建议把连击从“数字奖励”升级为“战斗温度计”：

| 连击 | 名称 | 反馈 | 玩法意义 |
|---|---|---|---|
| 10 | 灵机初动 | 小字 + 音效上扬 | 告诉玩家节奏已建立 |
| 30 | 势如破竹 | 攻击反馈增强 3 秒 | 鼓励继续压制 |
| 60 | 道韵奔流 | 道势获得 +30% 持续 5 秒 | 接近爆发窗口 |
| 100 | 万法将成 | 慢动作 0.3 秒，道势大幅增加 | 进入高潮 |
| 200 | 道之极致 | 全屏特写，奖励局内临时强化 | 稀有名场面 |

连击不建议直接给长期数值，以免滚雪球过猛；它更适合作为“快要高潮”的提示器。

## 16. 道势系统规格

`道势` 是把动作、天象、灵宠、构筑串起来的核心资源。它不能太复杂，也不能只是另一个怒气条。

### 资源规则

- 上限：100。
- 进入房间保留上一房 30%，Boss 房保留 60%。
- 脱战 5 秒后不衰减，避免玩家觉得被惩罚。
- 满 100 后进入“盈满”状态，持续 8 秒等待玩家释放；若不释放，自动触发“道法通明”。

### 获取表

| 行为 | 道势 |
|---|---:|
| 普通击杀 | +2 |
| 状态击杀 | +3 |
| 天象/地形击杀 | +5 |
| 一次攻击命中 3 个以上敌人 | +4 |
| 一次连锁击杀 5 个以上敌人 | +12 |
| 精英击杀 | +15 |
| Boss 阶段击破 | +20 |
| 完美闪避 0.25 秒内反击命中 | +8 |
| 灵宠协同命中 3 个以上敌人 | +10 |
| 道统觉醒 | 直接补满 |

### 消耗与爆发

| 爆发 | 触发 | 效果 | 设计用途 |
|---|---|---|---|
| 道法通明 | 满道势自动或短按 | 5 秒内攻速 +20%、法术 CD 回复加快、粒子强化 | 稳定短爽 |
| 万法归一 | 满道势长按/专用键 | 按当前最强谱系释放一次大爆发 | 构筑身份表达 |
| 逆命一息 | 濒死且道势 ≥ 80 | 消耗全部道势，免死 1 秒并击退周围敌人 | 死亡前反杀 |

### 万法归一按谱系变形

| 当前谱系 | 万法归一表现 |
|---|---|
| 焚天 | 角色脚下火环扩散，所有灼烧敌人引爆并传播火种 |
| 雷罚 | 天雷连续锁定高威胁敌人，积水区域额外导电 |
| 沧海 | 全屏水波减速，冻结积水，随后冰裂爆开 |
| 万毒 | 毒云向外铺开，毒杀敌人时治疗玩家 |
| 不动 | 生成护体山岳，吸收伤害后反震 |
| 五行 | 五色轮转，按敌人当前状态触发对应处决 |

这能让同一个资源在不同 Build 下有完全不同记忆点。

## 17. 三套首发核心构筑样板

先把三套做深，比做十套浅要更好。首发建议选火、雷、水冰，因为它们和当前数据表、天象、状态已有较好基础。

### 焚天道统：传播与爆燃

体验关键词：越烧越大，最后一发全屏连爆。

| 阶段 | 关键获得 | 玩家玩法变化 |
|---|---|---|
| 道胚 | 火焰弹/引燃 | 普攻和 Q 开始稳定灼烧 |
| 道脉 1 | 火借风势 | 弹道穿透，火可以从单体变成一线 |
| 道脉 2 | 爆燃 | 满层灼烧可引爆，玩家开始寻找聚怪机会 |
| 道脉 3 | 纯阳真火 | 爆燃伤害变成可靠清怪手段 |
| 道统 | 焚天 | 每次爆燃留下火场，火场击杀继续传播 |

核心操作链：

```text
聚怪 -> 普攻/Q 叠灼烧 -> E 穿透传播 -> R 或万法归一引爆 -> 火场继续烧
```

弱点与制衡：

- 单体 Boss 需要靠火场持续伤害，不应瞬秒。
- 雨天削弱直接火伤，但可产生蒸汽遮挡/蒸汽爆，给替代玩法。
- 火系过强时，优先削传播半径，不削爆燃手感。

### 雷罚道统：导电与斩首

体验关键词：看准水和高威胁目标，一道雷清掉战场。

| 阶段 | 关键获得 | 玩家玩法变化 |
|---|---|---|
| 道胚 | 雷刃/九霄雷引 | 有麻痹，有 E 槽雷法 |
| 道脉 1 | 连环闪电 | 雷可以弹射，清杂能力出现 |
| 道脉 2 | 连环雷诀 | E 变成三连穿透，玩家开始打角度 |
| 道脉 3 | 雷体 | 近身风险下降，贴水作战可行 |
| 道统 | 雷罚 | 周期落雷，积水中必定扩散 |

核心操作链：

```text
把怪引进积水 -> E 连环雷穿透 -> 麻痹定身 -> 天雷/道势补刀 -> 高威胁斩首
```

弱点与制衡：

- 空旷无水房间雷罚清场效率下降，但斩首仍强。
- 雷系应更依赖目标选择和地形判断。
- 雷系过强时，优先增加弹射衰减，不削首跳伤害。

### 沧海道统：冻结与碎冰

体验关键词：先控制整片战场，再用一次碎冰收割。

| 阶段 | 关键获得 | 玩家玩法变化 |
|---|---|---|
| 道胚 | 寒霜/寒冰诀 | 减速形成生存优势 |
| 道脉 1 | 玄冰残卷 | 提前获得 R 槽冰法 |
| 道脉 2 | 玄冰枪谱 | 单点高伤，能打精英 |
| 道脉 3 | 冰火两重天/水系护盾词条 | 冰控 + 爆发或防守成型 |
| 道统 | 沧海 | 冻结敌人被击杀时产生冰裂 |

核心操作链：

```text
站水边诱敌 -> R 冻结积水 -> Q/E 点杀精英 -> 冻裂连锁 -> 滑步撤离
```

弱点与制衡：

- 直接伤害低于火雷，但容错更高。
- Boss 不应被长时间冻结，只吃“破招冻结”或短硬直。
- 水冰过强时，削冻结时间，不削冰裂视觉和声音。

## 18. 战斗房深度模板

### 房间势参数

每个战斗房进入时抽 1 个 `room_momentum`。它不是新模式，而是给当前房间一个可读主题。

| room_momentum | 适配天象 | 核心规则 | 触发高潮 |
|---|---|---|---|
| thunder_pool | 雨/雷 | 积水区逐渐扩大，雷命中扩散 | 击杀配额 70% 后落一次大雷 |
| sun_crack | 火/沙暴 | 干燥裂地周期出现，火命中爆燃 | 裂地连爆 3 次后全场火浪 |
| wind_eye | 风 | 风眼在场中移动，顺风弹道加速 | 顺风命中 5 个敌人后风刃爆发 |
| demon_anchor | 任意 | 1-3 个阵眼强化敌人 | 打碎阵眼全场眩晕 + 道势 |
| spirit_spring | 雨/雪/clear | 灵泉治疗圈同时影响敌我 | 玩家占住 3 秒获得护盾 |
| greed_cache | 任意 | 场上宝匣吸引敌群 | 守住/打碎后给灵石或词条 |

### 一房三幕数值

以 120 秒、20 击杀配额的第 1 关为例：

| 幕 | 时间/进度 | 刷怪 | 事件 |
|---|---|---|---|
| 探势 | 0-15 秒 | 低密度，2-4 只 | 展示地形和房间势 |
| 聚势 | 15-75 秒 | 正常密度，4-6 只/波 | 玩家完成主要击杀 |
| 破势 | 击杀 70% 或剩余 25 秒 | 精英/阵眼/大天气触发 | 给一次爆屏机会 |
| 收势 | 完成配额后 2 秒 | 停止刷怪 | 结算、吸收灵气、出奖励 |

注意：破势不是单纯加难，而是给“可利用的大机会”。高压和机会要同时出现。

## 19. 敌人与精英职责打磨

### 普通敌人阵容

| 职责 | 当前可映射 archetype | 行为 | 玩家最佳应对 |
|---|---|---|---|
| 追猎者 | berserker/skirmisher | 持续贴脸，逼走位 | 聚怪后爆发 |
| 弹幕手 | ranged | 中距离散射 | 闪避切入或穿透反打 |
| 狙击者 | sniper | 长预警高伤直线 | 横移躲开，反用其站位聚怪 |
| 护阵者 | shaman | 给附近敌人护盾/加速 | 优先斩首 |
| 自爆魔 | bomber，可新增 | 低血冲锋爆炸 | 引到怪群中点爆 |
| 吞灵虫 | summoner，可新增 | 拖久召唤小虫 | 快速清掉，防滚雪球 |

### 精英设计公式

每个精英 = 一个普通职责 + 一个可读词缀 + 一个死亡奖励。

例子：

| 精英 | 行为 | 破解 | 死亡奖励 |
|---|---|---|---|
| 雷核护阵者 | 周期放电，水中更危险 | 引到怪群再杀 | 死亡导雷，麻痹全场 |
| 火甲追猎者 | 近身灼烧玩家 | 冰/水破甲 | 爆出火场 |
| 贪灵弹幕手 | 吸收灵石变强 | 越早杀越安全 | 掉更多灵石 |
| 镜影精英 | 复制玩家一个最高标签 | 用相克天象打 | 给对应道统进度提示 |

精英不能只是血厚。它必须让玩家改变优先级。

## 20. Boss 细化

### Boss 战三段结构

每个 Boss 都用固定节奏，降低制作复杂度：

1. **识招：** 展示核心招式，伤害低，预警明显。
2. **破招：** 加入可反制窗口，玩家可用天象/道势打断。
3. **渡招：** Boss 放大招，玩家用本关学到的技巧通过。

### 首个 Boss：赤焰守关傀儡

教学目标：让玩家第一次理解“地形不是背景”。

| 阶段 | Boss 行为 | 玩家机会 |
|---|---|---|
| 100%-70% | 冲锋爪击，留下短裂地 | 引导 Boss 冲过怪群 |
| 70%-35% | 砸地生成干燥裂纹 | 用火命中裂纹爆燃 |
| 35%-0% | 连续冲锋 + 小怪 | 聚怪，引爆裂地清小怪 |

击败演出：最后一块火纹核心碎裂，地面裂纹向外燃尽，大字“破炉”。

### 第二 Boss：幽泉秘境镇守

教学目标：水面可以导雷，也可以冻结。

| 阶段 | Boss 行为 | 玩家机会 |
|---|---|---|
| 100%-70% | 召水波，地面出现水道 | 雷法可扩散 |
| 70%-35% | 蓄力吞水回血 | 冰法冻结水道可打断 |
| 35%-0% | 水位上涨，小怪增多 | 冻结大水面后碎冰清场 |

### 第三 Boss：天雷劫影

教学目标：预警圈既是危险，也是武器。

| 阶段 | Boss 行为 | 玩家机会 |
|---|---|---|
| 100%-70% | 固定落雷圈 | 诱导落雷劈怪 |
| 70%-35% | 追踪落雷 | 闪避后让雷劈 Boss 影子 |
| 35%-0% | 三连天雷 | 满道势可牵雷反击 |

## 21. 奖励与选择打磨

### 三选一结构

每次词条选择都应尽量由三种张力组成：

| 位置 | 名称 | 内容 |
|---|---|---|
| A | 稳定补强 | 攻击、生命、防御、攻速、穿透 |
| B | 道路推进 | 当前最接近 Combo/道统的词条 |
| C | 偏门诱惑 | 高品质、代价、跨属性、灵宠/器灵相关 |

这样玩家每次都不是“哪个数值大”，而是在问：

```text
我现在要活？要成道？还是贪一手奇遇？
```

### 奖励节奏

| 房间 | 奖励重点 |
|---|---|
| 普通战斗 | 普通三选一，主要给道胚/基础补强 |
| 精英战斗 | 高品质词条，强推道脉 |
| 事件 | 风险换稀有方向、灵宠、因果 |
| 坊市 | 修正路线，购买缺口 |
| Boss | 境界突破 + 道统关键件高权重 |

Boss 后奖励一定要像“突破境界”，不要像普通掉落。建议 Boss 后固定有两步：

1. 境界天赋三选一。
2. 当前最佳道统谱系的关键词条/替代件三选一。

## 22. 难度与容错打磨

### 三种道心的真实差异

难度不只改血量和数量，要改“信息量与容错”：

| 道心 | 信息 | 容错 | 奖励 |
|---|---|---|---|
| 问道 | 天象提前提示，精英词缀解释更明显 | 完美闪避窗口更宽，灵宠救命更频繁 | 常规 |
| 悟道 | 标准提示 | 标准窗口 | 标准 |
| 证道 | 提示更少，房间势更激进 | 窗口更窄，敌人组合更毒 | 品质上限和心魔奖励更高 |

### 动态容错

为了保持“爽而不挫”，后台可以做轻量保护：

- 连续两房低血过关：下一房开局给一个小灵泉或调息事件权重提高。
- 连续两次死亡在第 1 关：下局首个道胚更贴近上一局最常用元素。
- 玩家连续 60 秒没有触发天象击杀：下一房生成更容易利用的地形布局。

这些不要明说，但会让玩家觉得“我越来越懂了”。

## 24. 调参指标

玩法是否变爽，需要用指标辅助判断：

| 指标 | 目标 |
|---|---|
| 每个普通战斗房明显高潮次数 | 1-2 次 |
| 玩家每局首次道势满时间 | 第 2-3 个战斗房 |
| 玩家每局首次 Combo 完成时间 | 第 1 关末或第 2 关初 |
| 玩家每局首次道统接近完成提示 | 第 3 关前 |
| 普通房平均时长 | 75-120 秒 |
| 精英房平均时长 | 90-150 秒 |
| Boss 战平均时长 | 90-180 秒 |
| 天象击杀占比 | 新手 5%-10%，熟练玩家 20%-35% |
| 万法归一每局释放次数 | 主线 3-6 次 |

如果玩家一局没有释放过万法归一，说明道势获取太少或 UI 引导不足。

如果玩家每房都自动触发满屏爆炸，说明道势获取太多或房间势触发太密。

## 25. 下一步建议

最建议的下一步不是做完整大系统，而是做一个“焚天切片”：

```text
1 个天象：烈阳
1 个房间势：烈阳裂地
1 套词条：火焰弹、引燃、爆燃、火借风势、纯阳真火、焚天
1 个道势爆发：焚天版万法归一
1 个 Boss 机制：冲锋留下干燥裂地，可被火引爆
```

只要这条链路打爽，雷罚、沧海、万毒都可以照这个模板复制扩展。

## 26. 开局合理性重构

当前原型开局默认“普攻发弹道 + Q 烈焰弹”，这对验证战斗系统很方便，但作为正式体验不合理：

- 所有玩家开局都像远程法修，削弱剑修、体修、符修、丹修、散修等身份差异。
- 功法像是凭空出现在快捷栏里，不像修仙世界中的传承、悟道、机缘。
- 开局没有“我是谁”的回答，导致后续词条和道统缺少根。
- 近战、护身、阵地、召唤、符箓等修仙战斗幻想没有进入第一分钟。

开局必须先回答三个问题：

```text
我是谁？我凭什么会这招？我这一世准备走哪条道？
```

### 新开局结构

开局从“选一个远程法术”改为四步：

1. **选出身/道途：** 决定基础攻击形态和第一套成长方向。
2. **领本命招式：** 开局就会的普通攻击，不一定是远程。
3. **得第一卷功法：** 明确功法来源，作为本局道胚。
4. **入第一场试炼：** 第一房用于让玩家理解自己的攻击距离和核心操作。

流程示例：

```text
道心选择 -> 道途选择 -> 前世遗泽/第一卷功法 -> 踏入轮回 -> 入门试炼
```

### 道途决定开局攻击形态

道途不是完整角色职业，可以先做成轻量“开局战斗模板”。以后再扩成角色/灵根系统。

| 道途 | 本命招式 | 攻击距离 | 开局 Q | 核心爽点 |
|---|---|---:|---|---|
| 剑修 | 近身剑气斩，第三击放短剑芒 | 近中 | 御剑突刺 | 贴身走位、穿怪斩杀 |
| 法修 | 元气弹/属性弹 | 远 | 烈焰弹/雷击印/寒霜弹三选一 | 稳定远程、属性构筑 |
| 体修 | 拳掌连击，蓄力震地 | 近 | 崩山劲 | 硬吃小伤、近身爆发 |
| 符修 | 投符，落地成小阵 | 中远 | 火符/雷符/冰符 | 阵地控制、提前布置 |
| 丹修 | 药气短波，命中回灵 | 中 | 毒雾丹/回春丹诀 | 续航、毒与药性 |
| 散修 | 破旧飞剑 + 随机小术 | 混合 | 随机残卷 | 不稳定但选择更多 |

第一版不需要六个全做。最推荐先做 3 个：

| 首发道途 | 为什么先做 |
|---|---|
| 剑修 | 解决“全是远程”的最大观感问题，提供近战爽点 |
| 法修 | 保留当前远程弹道原型，承接火/雷/冰构筑 |
| 符修 | 用较低成本做出“不是弹幕射击”的阵地玩法 |

### 本命招式不是功法

要区分三个概念：

| 概念 | 解释 | 游戏表现 |
|---|---|---|
| 本命招式 | 角色天生会的基础战斗动作 | 普攻形态，随道途决定 |
| 功法 | 可成长的修炼法门 | 词条/技能树，能升层 |
| 法术 | 功法衍生的主动招 | Q/E/R 或替换技能 |

因此开局“会攻击”不等于“凭空会功法”。剑修会挥剑，体修会挥拳，符修会画低阶符，这些是本命招式。真正的功法需要来源。

### 功法来源

功法必须从修仙语境来，且要让玩家看见。

| 来源 | 出现时机 | 玩法作用 | 示例 |
|---|---|---|---|
| 师承残篇 | 开局道途自带 | 保证基础构筑方向 | 剑修得《青锋诀残篇》 |
| 前世遗泽 | 非首局开局 | 继承上局未竟道路 | “残雷入骨”给雷系道胚 |
| 秘境机缘 | 战斗/事件奖励 | 局内主要获取 | 洞府、石碑、残卷 |
| 夺法/交易 | 事件/坊市 | 风险换强功法 | 魔修交易、坊市买残卷 |
| 临战顿悟 | 连击/天象/突破 | 让操作转化为成长 | 雷雨中连杀后悟《引雷诀》 |
| Boss 传承 | 关底 | 阶段性强奖励 | 击败幽泉镇守得水系秘卷 |

文案上，词条三选一不应叫“选择词条”，而是：

```text
此战灵机汇聚，你从三缕道韵中择其一：
```

或按来源变化：

```text
残卷浮现三行真意：
坊市摊主取出三卷残篇：
天雷落下，你顿悟三种雷意：
```

这样功法“来得合理”，奖励界面也更有修仙感。

## 27. 开局道途详细设计

### 剑修：近中距离、斩线、机动

定位：给玩家一个不是远程弹幕的标准开局。

本命招式：

- 普攻为三段剑斩。
- 第 1/2 段是近身扇形斩。
- 第 3 段放出短距离剑芒，可穿透 1 个敌人。
- 完美闪避后 1 秒内普攻变为“回身斩”，范围更大。

开局功法来源：

```text
师承残篇：《青锋诀残篇》
你这一世醒来时，掌心有一道旧剑痕。剑痕中藏着三式残诀。
```

开局 Q：

- `御剑突刺`：向目标方向短突进并斩击路径敌人。
- 如果命中 3 个以上敌人，返还部分 CD。

第一套成长：

| 层级 | 名称 | 效果 |
|---|---|---|
| 1 | 青锋 | 普攻第三段剑芒距离 +20% |
| 2 | 回风 | 完美闪避后回身斩必定暴击 |
| 3 | 剑气 | 剑芒可触发元素词条 |
| 4 | 御剑 | Q 突刺后留下飞剑 2 秒 |
| 5 | 剑心 | 连续斩杀时获得道势 |

剑修的关键爽点不是站远处射，而是：

```text
闪避穿过攻击 -> 回身斩 -> 第三段剑芒穿线 -> 御剑突刺收割
```

### 法修：远程属性、稳定构筑

定位：保留当前原型最成熟的远程法术，但补上“为什么会法术”。

本命招式：

- 普攻为低阶元气弹。
- 根据开局第一卷功法，元气弹可变为火、水、雷之一。

开局功法来源：

```text
轮回殿赐卷：《五行入门篇》
轮回殿只给你一卷入门残篇，真正的道要在局内自己走出来。
```

开局 Q 三选一：

| 选项 | 功法来源 | Q |
|---|---|---|
| 火 | 《小离火诀》 | 烈焰弹 |
| 雷 | 《引雷诀残页》 | 雷击印 |
| 水冰 | 《寒泉诀》 | 寒霜弹/玄冰刺 |

法修是唯一开局就可以纯远程的道途。这样“远程”从默认行为变成身份特色。

### 符修：中距离、阵地、预判

定位：让玩家一开始就体验“布置法门”，而不是只射弹道。

本命招式：

- 普攻投出纸符。
- 纸符命中敌人造成小伤；落空则贴地 2 秒。
- 贴地纸符被 Q/E/R 或敌人踩中会触发小爆。

开局功法来源：

```text
家传旧匣：《三才符解》
匣中只剩几张残符，符胆未失，可借灵气重绘。
```

开局 Q：

- `起符`：引爆附近纸符；若没有纸符，则向前放一枚火符。

成长方向：

| 层级 | 名称 | 效果 |
|---|---|---|
| 1 | 纸符 | 普攻落地符持续时间 +1 秒 |
| 2 | 连符 | 起符可连锁 2 张符 |
| 3 | 阵眼 | 三张符形成小阵，持续伤害 |
| 4 | 借势 | 天象地形中的符爆效果变化 |
| 5 | 万符 | 万法归一变为全场符阵齐爆 |

符修的爽点：

```text
边退边丢符 -> 怪群踩进符区 -> 起符连爆 -> 天象改变符阵属性
```

### 体修：近身承压、蓄力震荡

体修可以作为第二批，因为需要更细的受击、霸体和近战手感。

本命招式：

- 普攻拳掌连击。
- 按住攻击蓄力，松开震地。
- 闪避改为短冲肩撞，撞到敌人获得护体。

开局 Q：

- `崩山劲`：前方短距离冲拳，击退敌人，撞墙/撞障碍造成额外震荡。

核心成长：

- 受击不打断时叠“血气”。
- 血气满后进入短霸体。
- 道势爆发变成“金身不坏”，吸收伤害后反震。

## 28. 第一卷功法选择

开局给玩家的不是随机词条，而是“第一卷功法”。它决定第一局前 3 分钟的理解路径。

### 首局默认

首次游玩建议不要让玩家面对太多选项：

```text
道途：法修
第一卷：《小离火诀》
本命招式：元气弹
Q：烈焰弹
```

但第二局开始，开放道途选择。这样新手先学基础，随后马上知道“原来不是只能远程火弹”。

### 非首局开局

非首局流程：

1. 选择道心。
2. 选择道途。
3. 选择第一卷功法，3 选 1：
   - 道途本命卷。
   - 前世遗泽相关卷。
   - 随机机缘卷。

示例：

```text
剑修开局三选一：
1. 《青锋诀残篇》：强化剑芒与回身斩。
2. 《小离火诀》：剑斩附火，走焚天剑路。
3. 《引雷诀残页》：完美闪避后下一剑引雷。
```

这样功法来源、构筑方向和玩法变化在开局就统一起来。

## 29. 对当前原型的调整建议

### 需要改掉的默认设定

当前：

```text
普攻 = 远程弹道
开局 Q = 烈焰弹
E/R = 突破后默认解锁雷/冰远程法术
```

建议改为：

```text
普攻 = 由道途决定
开局 Q = 由第一卷功法决定
E/R = 通过突破、功法升层、词条、事件习得
```

### 最小实现路线

第一步只需要新增“攻击形态”概念，不必一次做完整角色系统。

新增配置：

```csv
path_id,name,basic_attack_type,start_manual,start_q,description
sword,剑修,melee_combo,qing_feng_manual,yu_jian_thrust,近身三段剑斩
caster,法修,projectile,xiao_li_huo,lie_yan_bolt,远程属性法术
talisman,符修,talisman_trap,san_cai_fu,qi_fu,投符布阵
```

代码层概念：

| 模块 | 变化 |
|---|---|
| `RunContext` | 记录 `starting_path_id`、`starting_manual_id` |
| `player.gd` | 普攻根据 `basic_attack_type` 分派 |
| `SpellProgress` | 默认绑定不再写死火/雷/冰，而从开局功法初始化 |
| `active_spells.csv` | 增加近战突刺、起符、寒霜弹等开局 Q |
| `affixes.csv` | 词条可以要求/强化某个道途 |

### 最小验证切片

先做两种开局就足够验证问题是否解决：

| 切片 | 内容 |
|---|---|
| 剑修切片 | 近战三段斩 + 御剑突刺 + 青锋诀 3 层 |
| 法修切片 | 当前远程弹道 + 烈焰弹 + 小离火诀 3 层 |

验收标准：

- 玩家开局 30 秒内能明显感到“剑修和法修完全不是一个距离”。
- 玩家能说出自己功法从哪里来。
- 奖励选择不再像凭空掉词条，而像战斗后领悟或获得残卷。
- E/R 的获得有叙事来源：突破、残卷、事件或词条，而不是自动塞进快捷栏。

## 30. 开局叙事与 UI 文案

开局界面建议从“选道心 + 踏入轮回”扩成“本世立道”。

界面结构：

```text
道心：问道 / 悟道 / 证道
道途：剑修 / 法修 / 符修
第一卷：青锋诀残篇 / 小离火诀 / 三才符解

踏入轮回

每个道途卡片要显示：

- 攻击距离：近 / 中 / 远。

- 第一卷功法来源：例如“师承残篇”。
- 推荐道统：例如“九世剑仙 / 焚天剑路”。

示例文案：

剑修

本命招式：三段剑斩

法修

本命招式：元气弹

符修

第一卷：三才符解
来源：破旧符匣中仅存的家传残篇
```

这样开局会从“系统给了我一个远程技能”变成“这一世我以什么身份入道”。

## 31. 玩法自洽性审计

除了“开局全是远程”和“功法来源不明”，当前还有一批更深的不合理点。它们不一定是工程 bug，但会让玩家潜意识里觉得世界不成立、成长不可信、选择不够有意义。

审计标准：

```text
这个系统在修仙世界里为什么存在？
玩家为什么能获得这个奖励？
这个选择是否有代价？
这个阶段是否真的对应境界变化？
玩法表现和文案承诺是否一致？
```

### 总览清单

| 优先级 | 问题 | 当前表现 | 玩家感受 | 修正方向 |
|---|---|---|---|---|
| P0 | 境界与关卡错位 | 第 3 关叫“渡劫前夕”，但境界还是金丹/元婴流程 | 修仙成长线混乱 | 重排五关命名和突破时机 |
| P0 | Combo/道统判定过浅 | 一个词条可能同时提供多个 required tags | 成道太随便，不像里程碑 | required 改成“类别槽/关键件” |
| P0 | 奖励来源不明 | 清怪后直接三选一功法/词条 | 像系统发牌，不像修仙机缘 | 按来源包装奖励：妖丹、残卷、悟道、传承 |
| P0 | 敌人生态重复 | 五关敌池高度复用，Boss 机制也通用 | 每关不像新境界 | 每关建立独立敌人生态和 Boss 课题 |
| P1 | 灵宠承诺与实现不符 | 核心文档说灵宠很重要，当前只有火萤且 Boss 后自动给 | 结缘没有仪式，宠物像外挂弹道 | 改成灵兽结缘事件，至少三种宠物可选 |
| P1 | 坊市/调息太像菜单按钮 | 战斗后随时继续/休息/进坊市 | 路线选择没有世界逻辑 | 坊市和调息成为房间/事件，不是每房通用按钮 |
| P1 | 突破像普通升级 | Boss 后选天赋、加槽位，但缺少瓶颈/雷劫/代价 | “突破”没有仪式和风险 | 加突破试炼、心魔问题或天劫段落 |
| P1 | 经济来源过游戏化 | 杀怪固定给灵石，跳过词条给灵石 | 灵石像金币，不像修仙资源 | 灵石来源改为妖丹炼化/矿脉/坊市交易 |
| P2 | 道心难度只像数值档 | 问道/悟道/证道主要影响难度奖励 | 道心不是心性选择 | 让道心改变信息、代价和因果倾向 |
| P2 | 前世遗泽过机械 | 从已有 affix 里选一个带到下局 | 像继承装备，不像轮回残响 | 改成“道痕/执念/残魂”三类遗泽 |
| P2 | 天象固定在关卡 | 每关固定 weather_id | 每局变化不足，天象像关卡皮肤 | 每房抽天象，按境界有权重和预告 |

## 32. 境界与关卡重排

### 当前问题

当前数据里五关是：

```text
1 初入仙途
2 秘境深处
3 渡劫前夕
4 焚心秘域
5 天劫试场
```

但运行时境界是：

```text
1 炼气
2 筑基
3 金丹
4 元婴
5 渡劫
```

问题在于：

- 第 3 关就叫“渡劫前夕”，节奏提前透支。
- “突破”只是加词条槽和给天赋，没有真正把玩家从一个境界推到下一个境界。
- Boss 是“关底守将”，但修仙语义上 Boss 应该是“瓶颈/劫/心魔/守道者”。

### 修正方案

每一关明确为一个境界阶段，每个 Boss 是该境界的瓶颈。击败 Boss 后才突破到下一境。

| 关 | 进入时境界 | 关卡名 | Boss 本质 | 击败后 |
|---|---|---|---|---|
| 1 | 炼气 | 炼气期 · 引气入体 | 炼气瓶颈/妖兽试胆 | 突破筑基 |
| 2 | 筑基 | 筑基期 · 道基初成 | 道基心魔/幽泉守基 | 突破金丹 |
| 3 | 金丹 | 金丹期 · 丹成有劫 | 金丹雷劫/夺丹魔影 | 突破元婴 |
| 4 | 元婴 | 元婴期 · 神魂出窍 | 元婴心魔/上古残魂 | 突破化神或渡劫前境 |
| 5 | 渡劫 | 渡劫期 · 九劫飞升 | 天劫化身 | 飞升/入无尽 |

如果仍想保持五境，建议最终是“渡劫 -> 飞升”，不要在第 4 关后再说“突破到渡劫”时缺少一个真正的新阶段。

### 突破的合理流程

Boss 后不应直接弹天赋三选一，而应是：

```text
Boss 残魂/劫云出现 -> 玩家选择突破方式 -> 承受代价 -> 获得境界天赋 -> 开启新槽位
```

突破方式三选一：

| 类型 | 效果 | 代价 |
|---|---|---|
| 稳固突破 | 获得防御/生命/容错类天赋 | 奖励较稳，无额外品质 |
| 冒险突破 | 获得高攻击/道势/暴击类天赋 | 下一关开局有心魔压迫 |
| 借势突破 | 根据当前天象/道统给专属天赋 | 需要满足某个条件，否则效果较弱 |

这样“突破”才像修炼抉择，而不是普通升级面板。

## 33. Combo 与道统判定不合理

### 当前问题

当前 Combo/道统大多通过 raw tags 判断，例如：

```text
fire|burn|combust
thunder|paralyze|chain
```

但一个词条可能同时带多个 tag。这样会出现：

- 一个词条就推进多个条件。
- Combo 像标签收集，不像功法链路。
- 道统觉醒太早、太轻，失去“成道”的重量。

例如“火焰弹”自带 `fire|burn`，“爆燃”自带 `fire|combust`，两张就能完成 `fire|burn|combust`。这作为小 Combo 可以，但作为“焚天道统”不够。

### 修正方案：条件从 tag 改成槽位

道统条件应是多个“类别槽”，每个槽只能由特定类型填充。

焚天道统示例：

| 槽位 | 可满足内容 |
|---|---|
| 火源 | 火焰弹 / 小离火诀 / 火符 |
| 燃烧 | 引燃 / 灼烧强化 / 火毒 |
| 引爆 | 爆燃 / 炎爆符 / 焚炉诀 |
| 传播 | 火借风势 / 火场蔓延 / 飞剑带火 |
| 纯化 | 纯阳真火 / 焚天 / 烈阳借势 |

只有五个槽位都满足，才是焚天道统。

数据上可以仍保留 tags，但道统表不要只写 `required_tags`，而写：

```csv
id,name,slot_fire_source,slot_burn,slot_detonate,slot_spread,slot_purify
furnace_dao,焚天道统,fire_source,burn,detonate,spread,purify
```

或者更通用：

```csv
dao_id,slot_id,accepted_tags,required_count
furnace_dao,fire_source,fire_basic|fire_manual,1
furnace_dao,burn,burn|ignite,1
furnace_dao,detonate,combust|explode,1
furnace_dao,spread,spread|wind_fire,1
furnace_dao,purify,yang|furnace,1
```

### 小 Combo 与大道统分层

建议分三层：

| 层级 | 条件 | 作用 |
|---|---|---|
| 小联动 | 2 个条件 | 提供即时爽点，如爆燃/导雷 |
| 道脉 | 3 个类别槽 | 改变玩法循环 |
| 道统 | 5 个类别槽 + 行为条件 | 称号、演出、终局强化 |

行为条件可以是：

- 用爆燃击杀 30 个敌人。
- 雷雨天导电击杀 20 个敌人。
- 冻结精英 3 次。
- 完美闪避后斩杀 Boss 阶段。

这会让“成道”来自玩法行为，而不只是抽卡。

## 34. 奖励来源与事件不合理

### 当前问题

事件表中很多选项本质是：

```text
拿钱 / 回血 / 随机词条 / 离开
```

比如“神秘老者”给灵石或回血，“灵石矿脉”给灵石，“残卷机缘”给随机词条。这能跑通原型，但修仙味不强。更大的问题是：战斗清场后也直接给词条，玩家不知道自己为什么领悟。

### 修正原则

奖励必须有来源：

| 来源 | 对应奖励 | 文案逻辑 |
|---|---|---|
| 妖兽 | 妖丹、精血、兽骨 | 炼化成属性强化或灵石 |
| 修士敌人 | 残卷、法器、符箓 | 拾取/参悟/拆解 |
| 天象 | 道韵、灵机 | 借势顿悟 |
| 地形 | 灵泉、矿脉、古阵 | 探索获得 |
| Boss | 传承、瓶颈突破 | 阶段性大成长 |
| 坊市 | 交易、情报、丹药 | 用灵石换确定性 |

### 清场奖励改写

不要统一叫“清场奖励：灵石 +X”。按敌人和房间来源变化：

| 房间 | 奖励说法 |
|---|---|
| 妖兽潮 | “炼化妖丹，得灵石 +X” |
| 散修战 | “搜得残符碎卷，得灵石 +X” |
| 傀儡阵 | “拆解机关核心，得灵石 +X” |
| 天劫房 | “收束劫灰，得天道灵砂 +X” |

词条奖励也改为：

```text
此战道韵凝结，你可参悟一缕真意
妖丹中残留三种灵机
敌修残卷展开三行秘法
```

## 35. 敌人生态不合理

### 当前问题

当前每关敌人池高度复用：

```text
wild_wolf / sky_bat / mud_serpent / wind_mantis / furnace_golem
```

到第 5 关仍然主要是这些敌人，只是数值变高。这会造成：

- 境界提升没有新世界感。
- Boss 和小怪生态没有关系。
- 敌人行为不像“修仙世界的试炼”，像通用怪物池。

### 修正方案：每关一个生态主题

| 关 | 敌人生态 | 普通敌 | 精英 | 生态玩法 |
|---|---|---|---|---|
| 炼气 | 山野妖兽 | 妖狼、毒蝠、泥蛇 | 妖兽头领 | 教移动和基础状态 |
| 筑基 | 散修/宗门试炼 | 剑修、符修、丹奴 | 阵法师、堕修士 | 教优先级和阵地 |
| 金丹 | 魔染洞天 | 魔虫、魔修弟子、噬灵兽 | 魔将、噬灵母虫 | 教召唤和腐化地形 |
| 元婴 | 上古遗迹 | 傀儡、机关兽、残魂 | 机关巨像、怨灵 | 教机制敌和破阵 |
| 渡劫 | 天劫生灵 | 雷灵、火灵、风劫影 | 天劫先锋 | 教天象综合运用 |

这样每关的敌人不只是换皮，而是对应新的战斗问题。

### Boss 与生态绑定

每个 Boss 应该是本关生态的凝结：

- 炼气 Boss 是妖兽王，不应该用“关底守将”通用名。
- 筑基 Boss 是道基心魔/幽泉守基，技能应围绕水和筑基。
- 金丹 Boss 是夺丹魔影，应该会偷取/污染玩家构筑。
- 元婴 Boss 是上古残魂，应该有魂体分身和机关机制。
- 渡劫 Boss 是天劫化身，才使用雷罚/九劫。

如果 Boss 名字是“幽泉秘境镇守”，但技能仍是“劫火四起/天罚降世”，玩家会立刻觉得不对。

## 36. 灵宠系统不合理

### 当前问题

设计文档把灵宠放在核心体验层，但当前原型只有 `火萤`，且第 1 个 Boss 后如果没有宠物就自动获得。这有三个问题：

- 结缘没有选择和仪式。
- 宠物来源不明。
- 宠物行为仍像额外远程弹道，没有“战友感”。

### 修正方案：灵兽结缘不是掉落

第一次灵宠应来自一个明确事件：

```text
Boss 残魂散去，困在阵中的小灵兽显形。
它没有立刻臣服，而是在观察你此战所走之道。
```

结缘三选一：

| 选择 | 获得 | 条件倾向 |
|---|---|---|
| 以火灵温养 | 火灵雀/火萤 | 火系/焚天倾向 |
| 以雷意唤醒 | 雷灵貂 | 雷系/雷罚倾向 |
| 以清泉疗伤 | 水灵蛟/水灵鱼 | 水冰/生存倾向 |

如果首版资源不足，可以仍只有火萤，但要包装成：

- 第 1 关固定事件“灵萤认主”。
- 火萤不是强宠，而是引导灵宠系统的新手伙伴。
- 后续灵宠通过事件/坊市/巢穴扩展。

### 宠物要有非远程功能

每只宠物至少承担一个非伤害职责：

| 宠物 | 非伤害职责 |
|---|---|
| 火灵雀/火萤 | 点燃干燥地形，制造火种 |
| 雷灵貂 | 标记高威胁敌人，触发导电 |
| 水灵蛟 | 生成短水面/护盾 |
| 土灵龟 | 挡弹幕/救命 |
| 木灵蝶 | 毒雾/回血区 |

如果宠物只是“多射几发”，会继续加重“全是远程”的问题。

## 37. 坊市、调息与路径选择不合理

### 当前问题

战斗后路径选择里常见：

```text
继续深入 / 调息片刻 / 踏入坊市
```

如果每场战斗后都能休息或进坊市，会导致：

- 坊市不像地点，像菜单。
- 调息没有机会成本。
- 路线选择不再是 Roguelite 的路线规划，而是战后按钮。

### 修正方案

坊市、调息、事件都应该是房间节点，或至少是有限次数的路径分支。

战斗后可以有：

```text
吸收道韵 -> 选择下一路
```

下一路才是：

| 路径 | 意义 |
|---|---|
| 妖气更盛 | 精英/高奖励 |
| 灵气平稳 | 普通战斗/稳定 |
| 人声隐现 | 坊市/事件 |
| 古阵微光 | 修炼/隐藏 |
| 清泉潺潺 | 调息/恢复 |

调息也要有代价：

- 消耗一次房间机会。
- 降低下一次奖励品质。
- 或清除当前道势，表示心神归静。

不然“受伤就休息”会把战斗压力磨平。

## 38. 经济与灵石不合理

### 当前问题

灵石目前承担了金币功能：清场给、事件给、跳过词条给、坊市花。它可用，但缺少修仙世界解释。

尤其是“跳过词条获得灵石”容易让人出戏：为什么我拒绝领悟，反而凭空得到钱？

### 修正方案

跳过词条改成三种选项之一：

| 行为 | 结果 | 文案 |
|---|---|---|
| 参悟 | 获得词条/功法 | “纳此道韵入身” |
| 炼化 | 获得灵石/道势 | “将道韵炼成灵砂” |
| 舍弃 | 回复/净化负面 | “散去杂念，稳固心神” |

这样灵石不是凭空奖励，而是你把机缘炼化成资源。

坊市也应区分货币：

- 灵石：通用交易。
- 妖丹：换丹药/宠物。
- 残卷：换功法。

首版不用真的做三货币，但文案和来源可以先区分。

## 39. 前世遗泽不合理

### 当前问题

死亡时从已有 affix 里选一个带到下局，这在 Roguelite 上成立，但在修仙语义上略机械。

问题：

- 为什么一个具体词条能完整带到下一世？
- 如果带的是“攻击强化”这种普通数值，很没有轮回感。
- 继承会不会破坏开局道途选择？

### 修正方案

前世遗泽不要直接继承词条，而是继承“道痕”。

| 遗泽类型 | 来源 | 下局效果 |
|---|---|---|
| 道痕 | 本局最高进度道统 | 提高相关道胚权重 |
| 执念 | 死亡前未完成目标 | 开局获得一个带代价的追求 |
| 残魂 | 名场面行为 | 给一次性强触发 |
| 旧缘 | 灵宠/事件关系 | 开局更容易遇到相关事件 |

示例：

```text
焚天未成 · 道痕
下局前 2 关火源/引爆类功法权重 +25%，但水系奖励权重 -10%。
```

这样继承的是“上一世的执念方向”，不是复制装备。

## 40. 道心难度不合理

### 当前问题

问道/悟道/证道是非常好的包装，但如果只影响敌人数量、HP 或奖励品质，它本质还是简单/普通/困难换名。

### 修正方案

道心应影响三件事：

| 道心 | 信息 | 代价 | 诱惑 |
|---|---|---|---|
| 问道 | 更多预告、更多解释 | 奖励较稳 | 容易完成基础道统 |
| 悟道 | 标准信息 | 标准代价 | 平衡 |
| 证道 | 少预告、更多心魔 | 死亡压力大 | 更高品质、更快成道 |

事件也应按道心改变选项：

```text
问道：多一个“稳妥参悟”选项。
悟道：标准三选一。
证道：多一个“逆天强取”选项，强但加心魔/因果。
```

这样道心才是修士心性，而不是难度按钮。

## 41. 天象随机性不合理

### 当前问题

当前每个 stage 有固定 weather_id。这对原型可控，但和“每局天象变化、天地为棋”的核心卖点冲突。

如果第 2 关永远雨、第 3 关永远雷，玩家很快会把天象当作关卡固定皮肤。

### 修正方案

天象应按房间抽取，但受境界和路线影响：

| 境界 | 天象策略 |
|---|---|
| 炼气 | 多晴/小雨，只出现一个简单交互 |
| 筑基 | 引入雨、烈阳、风 |
| 金丹 | 雷、雾、雪开始出现 |
| 元婴 | 一关内多次变化 |
| 渡劫 | 每房天象都强，Boss 使用多天象 |

同时保留“预告”：

```text
下一路妖风渐起
远处雷云压境
地脉燥热，似有烈阳之兆
```

玩家选择路线时就能把天象纳入策略。

## 42. 自动战斗辅助的不合理风险

### 当前问题

自动瞄准/自动普攻很适合降低门槛，也方便测试。但如果核心爽点来自“聚怪、借势、引爆、闪避”，自动战斗可能把游戏变成边走边挂机。

### 修正方案

自动功能可以保留，但要定位为“辅助”，不是最优玩法。

| 功能 | 保留原因 | 限制 |
|---|---|---|
| 自动瞄准 | 降低瞄准负担 | 不自动选择天象地形最佳角度 |
| 自动普攻 | 降低疲劳 | 不触发完美手动连段奖励 |
| 手动施法 | 核心操作 | 享受道势/完美释放奖励 |

建议增加“手动精彩奖励”：

- 手动 E 穿透 3 个以上敌人：额外道势。
- 手动 Q 命中正在爆燃/导电区域：额外反馈。
- 完美闪避后手动反击：额外道势。

这样自动辅助不会破坏上限。

## 43. 新总原则

后续所有系统都按这几条检查：

1. **来源原则：** 功法、灵宠、灵石、道统都要有世界内来源。
2. **身份原则：** 开局道途必须改变基础攻击距离和战斗节奏。
3. **境界原则：** 每一关的敌人、Boss、奖励必须服务当前境界主题。
4. **代价原则：** 强奖励必须有代价或条件，不要白给。
5. **行为原则：** 成道不只靠抽到词条，还要靠玩家做过对应行为。
6. **生态原则：** 敌人不是数值容器，每关要有自己的生态和战斗问题。
7. **仪式原则：** 突破、结缘、觉醒、飞升都要有独立仪式感。

如果只能先改三件事，优先级是：

```text
1. 开局道途 + 本命招式，解决身份问题。
2. 境界/关卡/Boss 重排，解决主线骨架问题。
3. Combo/道统条件重做，解决成道太轻的问题。
```

## 44. 肉鸽爽感与随机性不能丢

前面的重构强调“合理性、来源、身份、境界”，但这不能把游戏做成严肃模拟器。它首先仍然是肉鸽动作游戏，必须保证：

```text
每局都有新东西
每分钟都有爽点
每次死亡都想再试一把
每次成型都觉得这一局独一无二
```

修仙设定负责让随机结果“来得合理”，但随机性本身必须足够大胆。玩家不是来背设定的，而是来体验“这一世我居然这么成道”的。

### 随机性的三层结构

随机不能只是奖励三选一。建议拆成三层：

| 层级 | 随机内容 | 作用 |
|---|---|---|
| 局前随机 | 第一卷功法、前世遗泽、初始天象倾向、心魔种子 | 让开局方向不同 |
| 局中随机 | 房间路线、敌群组合、天象、奇遇、Boss 变体、奖励来源 | 让过程不同 |
| 成型随机 | 道统组合、铭纹、法宝、器灵觉醒、代价型强奖励 | 让 Build 不同 |

玩家可以有清晰身份，但不能只有一条固定成长线。剑修这一局可以是雷剑，下一局可以是血剑，再下一局可能是风雪飞剑流。

### 爽点节奏

肉鸽爽感要按时间密度设计。

| 时间尺度 | 玩家爽点 | 示例 |
|---|---|---|
| 1-3 秒 | 打击、闪避、爆炸、清屏 | 一剑斩出剑芒，雷链跳 5 个敌人 |
| 10-20 秒 | 小循环成立 | 聚怪、引爆、道势上涨、完美反击 |
| 1-2 分钟 | 构筑开始变形 | 青锋剑铭雷，普攻开始导电 |
| 3-5 分钟 | Build 第一次质变 | 三个雷系条件满足，触发雷剑道统 |
| 一整局 | 独特故事 | “这局被心魔逼着走成了魂雷剑修” |

如果玩家 30 秒内只是在绕圈打小怪，说明爽点密度不够。如果玩家 5 分钟内没有一次“构筑突然活了”的体验，说明肉鸽随机性不够。

### 随机要强，但要可解释

随机奖励不应该是系统凭空发牌，而是由来源包装：

| 随机事件 | 世界内解释 | 玩家感受 |
|---|---|---|
| 清怪掉残卷 | 妖修携带残破功法 | 我夺来了一门法 |
| 雷暴房给雷铭纹 | 天雷劈开器胚 | 我借到了天势 |
| Boss 掉法宝碎片 | 守关者本来就持有法宝 | 我夺了它的机缘 |
| 心魔给强词条 | 以因果换力量 | 我在赌命 |
| 前世遗泽改变奖励池 | 上一世执念未散 | 我在续前缘 |

核心原则：

```text
随机结果可以很疯，但来源不能很假。
```

### 随机池必须允许越界组合

如果每个道途只能拿本系奖励，游戏会很快变窄。应该允许“合理越界”：

| 本体 | 越界奖励 | 形成流派 |
|---|---|---|
| 剑修 | 雷击木铭纹 | 雷剑导电流 |
| 剑修 | 血契残页 | 血剑燃命流 |
| 法修 | 镇魂幡碎片 | 火法召魂流 |
| 符修 | 风系天象 | 风符扩散流 |
| 体修 | 玄武甲 | 反震不死流 |
| 丹修 | 魂晶 | 毒魂炼化流 |

道途给根，随机给枝。根要清楚，枝要野。

### 每局必须有 1-2 个“破格机缘”

为了让肉鸽足够爽，每局不能只有稳定成长，还需要少量破格事件。破格机缘不是白送强度，而是带条件、代价或风险的强奖励。

示例：

| 机缘 | 强度 | 代价 |
|---|---|---|
| 逆天铭纹 | 本命器立刻获得跨系效果 | 下一关天劫提前 |
| 心魔交易 | 获得高阶功法残页 | 心魔敌权重提高 |
| 上古器冢 | 三选一法宝胚 | Boss 获得对应抗性 |
| 妖丹强炼 | 立即突破一层祭炼 | 最大生命暂降 |
| 前世执念 | 指定奖励池偏向 | 其他属性奖励减少 |

这类事件的目的不是平衡，而是制造“这局不一样”的记忆点。

### 奖励三选一要更像肉鸽

普通奖励卡建议保持三类结构：

```text
A. 稳定增强：低风险，补短板。
B. 构筑推进：需要已有条件，帮助成型。
C. 破格诱惑：强、稀有、有代价，可能改变路线。
```

例子：剑修雷雨房清场后。

| 选项 | 类型 | 效果 |
|---|---|---|
| 回风剑势 | 稳定增强 | 闪避后下一剑范围扩大 |
| 雷击木铭纹 | 构筑推进 | 青锋剑第三段剑芒可导电 |
| 引劫入剑 | 破格诱惑 | 立刻获得高阶雷剑词条，但下一房天雷追身 |

这样玩家每次选择都有欲望，而不是只选数字最大。

### 随机性检查表

每新增一个系统，都用以下问题检查：

1. 它能否让两局同一道途玩出不同路线？
2. 它是否会在 1-2 分钟内给玩家一次明显变化？
3. 它有没有至少一个高风险高回报选项？
4. 它的奖励来源是否能被世界观解释？
5. 它是否允许跨属性、跨武器、跨天象组合？
6. 它是否会产生玩家愿意复述的故事？

如果答案大多是否，这个系统就只是设定，不是肉鸽玩法。
