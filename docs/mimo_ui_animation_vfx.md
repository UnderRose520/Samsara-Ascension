# 《轮回仙途》动画与特效系统设计文档

> **版本**: v1.0  
> **美学定位**: 暗色水墨风格 × 高饱和粒子爆发  
> **核心哲学**: 水墨流动为骨，金石铭文为魂，粒子爆发为血

---

## 一、全局动画语言

### 1.1 水墨流动缓动曲线 (Ink Flow Easing)

| 曲线名称 | 数学描述 | 适用场景 | 视觉感受 |
|---------|---------|---------|---------|
| `inkEaseIn` | `cubic-bezier(0.6, 0.0, 1.0, 0.4)` | 墨色渗入、能量聚集 | 如墨滴入水，缓慢凝聚后急速扩散 |
| `inkEaseOut` | `cubic-bezier(0.0, 0.6, 0.4, 1.0)` | 效果消散、界面退出 | 如毛笔离纸，先重后轻收尾 |
| `inkEaseInOut` | `cubic-bezier(0.7, 0.1, 0.3, 0.9)` | 场景转场、卡牌翻转 | 完整的笔锋起落过程 |
| `brushStroke` | `cubic-bezier(0.2, 0.8, 0.3, 1.0)` | 技能释放、攻击动作 | 笔触从粗到细，力量感递减 |
| `sealStamp` | `cubic-bezier(0.8, 0.0, 0.2, 1.0)` | 印章落下、暴击触发 | 突然冲击，然后缓缓沉降 |

**实现要点**:
- 所有缓动曲线在 60fps 下保持平滑，关键帧间隔不超过 16ms
- 水墨扩散使用径向渐变叠加，非线性alpha衰减
- 笔触效果通过贝塞尔曲线插值生成路径，配合压力感应模拟

### 1.2 笔触时序 (Brush Stroke Timing)

```
基础节拍: 1拍 = 0.33秒 (约20帧@60fps)
短笔触: 1拍起，2拍收 (0.33-0.66秒)
长笔触: 1拍起，3拍运，2拍收 (1.0秒)
顿笔: 1拍起，2拍顿，1拍收 (0.66秒)
飞白: 1拍起，快速掠过 (0.2秒内完成)
```

**节奏组合**:
- 普通攻击: 顿笔-顿笔-长笔触 (3连击，共1.66秒)
- 技能释放: 飞白-长笔触-墨爆 (蓄力-释放-爆发，共1.5秒)
- 受击反馈: 短笔触-墨溅 (0.5秒)

### 1.3 粒子哲学 (Particle Philosophy)

**核心原则**: 暗底高饱和

| 背景色阶 | 推荐粒子饱和度 | 推荐粒子明度 | 适用场景 |
|---------|---------------|-------------|---------|
| #0A0A0F (深墨) | 90-100% | 60-80% | 战斗主场景 |
| #1A1A2E (暗夜) | 80-95% | 50-70% | 菜单/过渡 |
| #2D2D3D (灰墨) | 70-85% | 40-60% | 天气/氛围 |

**色彩策略**:
- 主色调: 墨黑(#0A0A0F) + 朱砂红(#E63946) + 金箔(#FFD700)
- 元素色: 木翠(#2D6A4F) / 火焰(#FF6B35) / 金芒(#F4D03F) / 水蓝(#48CAE4) / 土褐(#BC6C25)
- 粒子始终从暗背景"跳"出来，不与背景融合

**粒子混合模式优先级**:
1. `Add` (叠加) - 发光效果、能量粒子
2. `Screen` (滤色) - 火焰、光效
3. `Multiply` (正片叠底) - 水墨晕染
4. `Alpha Blend` (标准) - 实体碎片

### 1.4 镜头震动模式 (Screen Shake Patterns)

| 震动类型 | 强度 | 持续时间 | 频率 | 衰减曲线 | 触发场景 |
|---------|------|---------|------|---------|---------|
| `hitLight` | ±2px | 0.15秒 | 60Hz | 线性衰减 | 普通命中 |
| `hitHeavy` | ±5px | 0.25秒 | 50Hz | 指数衰减 | 暴击/重击 |
| `hitMassive` | ±10px | 0.4秒 | 40Hz | 弹簧阻尼 | 范围技能/BOSS攻击 |
| `deathShake` | ±15px→±3px | 1.0秒 | 30Hz | 长尾衰减 | 玩家死亡 |
| `breakthrough` | ±8px→0 | 0.6秒 | 25Hz | 震荡衰减 | 境界突破 |
| `dodge` | ±1px | 0.1秒 | 80Hz | 瞬时消失 | 闪避动作 |

**震动叠加规则**:
- 同时触发多个震动时，取最大强度值，时长叠加
- 镜头震动不与镜头移动同时进行（移动优先）
- 暂停状态下冻结震动偏移量

## 二、UI过渡动画

### 2.1 场景转场 (Scene Transitions)

#### 2.1.1 主菜单 → 创建角色 (menu_to_setup)

```
时长: 1.2秒
阶段:
  [0-0.4s] 水墨晕染扩展 - 从屏幕中心向外扩散半透明墨色圆环
  [0.3-0.8s] 粒子飞散 - 金色粒子从中心四散，模拟墨滴溅落
  [0.6-1.2s] 新场景淡入 - 目标场景从模糊到清晰，配合粒子消散
  
特效层:
  - 背景: 墨色圆环径向扩展，边缘羽化
  - 前景: 30-50个金色小粒子，随机方向，寿命0.8秒
  - 覆盖层: 半透明黑色幕帘从四角向中心合拢再打开
  
缓动: inkEaseInOut
```

#### 2.1.2 创建角色 → 战斗 (setup_to_combat)

```
时长: 1.5秒
阶段:
  [0-0.3s] 角色立绘墨化 - 角色轮廓溶解为水墨粒子
  [0.2-0.6s] 墨色漩涡 - 屏幕中央形成旋转墨池
  [0.5-1.0s] 战斗场景从墨池中"浮现"
  [0.8-1.5s] 界面元素依次滑入
  
特效层:
  - 角色: 轮廓边缘粒子化，向中心收缩
  - 漩涡: 8-12层同心圆，旋转速度递减
  - 浮现: 战斗场景alpha 0→1，scale 0.9→1.0
  
缓动: brushStroke起始 + inkEaseOut收尾
```

#### 2.1.3 战斗 → 道法选择 (combat_to_choice)

```
时长: 0.8秒
阶段:
  [0-0.2s] 战斗画面快速模糊 (高斯模糊 0→10px)
  [0.1-0.4s] 三张卡牌从屏幕底部飞入，呈扇形展开
  [0.3-0.6s] 卡牌停止时产生轻微墨溅效果
  [0.5-0.8s] 卡牌光晕稳定
  
特效层:
  - 模糊: 线性插值
  - 卡牌: 各有独立轨迹，贝塞尔曲线入场
  - 墨溅: 每张卡牌落地时溅出5-8个小墨点
  
缓动: cardSlide (cubic-bezier(0.1, 0.9, 0.2, 1.0))
```

#### 2.1.4 道法选择 → 战斗继续 (choice_to_combat)

```
时长: 0.6秒
阶段:
  [0-0.2s] 选中卡牌放大+发光
  [0.1-0.3s] 其余卡牌墨化消散
  [0.2-0.5s] 选中卡牌化为粒子涌入玩家
  [0.4-0.6s] 战斗画面恢复清晰
  
缓动: inkEaseIn
```

### 2.2 面板动画 (Panel Animations)

#### 2.2.1 面板开启 (panel_open)

```
时长: 0.4秒
动画:
  - 面板从中心scale 0.8→1.0，alpha 0→1
  - 背景叠加半透明墨色蒙版 (alpha 0→0.6)
  - 面板边缘产生水墨晕染扩散效果
  
缓动: sealStamp
粒子: 面板边框出现时，沿边框流淌4-6个小型墨滴粒子
```

#### 2.2.2 面板关闭 (panel_close)

```
时长: 0.3秒
动画:
  - 面板scale 1.0→0.9，alpha 1→0
  - 背景蒙版alpha 0.6→0
  
缓动: inkEaseIn
粒子: 面板消失位置留下短暂墨迹残影 (0.2秒消散)
```

#### 2.2.3 卡牌滑入 (card_slide_in)

```
时长: 0.5秒
动画:
  - 卡牌从指定方向 (上/下/左/右) 滑入
  - 滑入过程中轻微旋转 (-5°→0°)
  - 到达位置时产生墨点溅射
  
缓动: cardSlide
粒子: 卡牌路径上留下3-5个半透明墨迹拖尾
```

#### 2.2.4 按钮状态 (button_states)

```
常态 (idle):
  - 边框: 1px 墨色描边
  - 背景: 半透明深灰
  
悬停 (hover):
  - 过渡: 0.15秒
  - 边框: 1px → 2px，颜色变为对应元素色
  - 背景: 亮度+10%
  - 粒子: 按钮边缘产生2-3个缓慢漂浮的墨滴
  
按下 (pressed):
  - 过渡: 0.05秒
  - scale: 1.0 → 0.95
  - 背景: 亮度-10%
  
释放 (released):
  - 过渡: 0.2秒
  - scale: 0.95 → 1.02 → 1.0 (弹簧效果)
  - 粒子: 从按钮中心向外扩散5-8个粒子
```

## 三、战斗特效 (Combat VFX)

### 3.1 伤害数字 (Damage Numbers)

#### 3.1.1 普通伤害

```
样式: 白色数字，黑色描边
动画:
  - 出现: scale 0→1.2→1.0，alpha 0→1
  - 漂浮: 向上移动30px，速度递减
  - 消失: alpha 1→0，scale 1.0→0.8
总时长: 0.8秒
位置: 伤害来源上方随机偏移 ±15px
缓动: inkEaseOut
```

#### 3.1.2 暴击伤害 (天机一击)

```
样式: 金色数字(#FFD700)，外发光，毛笔字体
动画:
  - 出现: 从0急速放大到1.5→1.0，伴随墨溅效果
  - 停留: 0.3秒
  - 漂浮: 向上移动50px
  - 消失: 数字边缘墨化溶解
总时长: 1.2秒
特效:
  - 数字周围环绕3-5个金色粒子
  - 出现时屏幕轻微震动 (hitHeavy)
  - 数字下方产生墨溅涟漪 (半径30px，0.3秒消散)
```

#### 3.1.3 治疗数字

```
样式: 翠绿色(#2D6A4F)，白色描边
动画:
  - 出现: 从下方升起，alpha渐显
  - 漂浮: 向上移动40px
  - 消失: 溶解为绿色粒子
总时长: 0.9秒
特效: 数字周围环绕小型叶片粒子 (2-3个)
```

### 3.3 技能特效

#### Q技能: 基础道法

```
释放动画 (0.6秒):
  [0-0.2s] 手部施法动作，墨色能量聚集在手心
  [0.2-0.4s] 能量凝聚为弹道，发射
  [0.4-0.6s] 命中后墨溅扩散

弹道特效:
  - 形态: 椭圆形墨团，边缘羽化
  - 粒子: 核心8-10个高亮粒子 + 外围15-20个半透明拖尾
  - 轨迹: 贝塞尔曲线，可被自动瞄准微调
  - 速度: 初始300px/s，命中时减速至0

命中特效:
  - 墨溅: 12-15个碎片向外扩散
  - 涟漪: 圆形冲击波，半径20-40px
  - 残留: 命中点留下0.3秒墨迹
```

#### E技能: 范围道法

```
释放动画 (0.8秒):
  [0-0.3s] 双手结印，地面出现法阵纹路
  [0.3-0.6s] 法阵扩展，能量柱升起
  [0.6-0.8s] 能量柱爆发，范围伤害

法阵特效:
  - 底层: 八卦图案，旋转速度15°/秒
  - 中层: 墨色能量漩涡，逆时针旋转
  - 顶层: 符文光点，沿法阵边缘流动
  
爆发特效:
  - 冲击波: 从中心向外扩散，半径80-120px
  - 能量柱: 高度150px，持续0.4秒
  - 粒子雨: 范围内落下30-50个能量碎片
```

#### R技能: 终极道法

```
释放动画 (1.2秒):
  [0-0.4s] 全屏暗化，角色悬浮，墨色能量环绕
  [0.4-0.8s] 能量凝聚为巨型墨龙/凤凰/玄武 (随机)
  [0.8-1.2s] 神兽冲锋，全屏特效

神兽特效 (以墨龙为例):
  - 身体: 8-10节墨色节段，每节独立运动
  - 龙鳞: 金色高光粒子，沿身体流动
  - 龙息: 前方扇形区域，持续0.4秒
  - 尾迹: 飞过路径留下墨色云雾，1秒消散
  
全屏效果:
  - 镜头: 跟随神兽移动
  - 震动: hitMassive
  - 粒子: 100+个能量粒子同时爆发
```

#### V技能: 宠物道法

```
释放动画 (0.5秒):
  [0-0.2s] 宠物发光，脱离玩家
  [0.2-0.4s] 宠物冲锋/施法
  [0.4-0.5s] 回归玩家身边

宠物特效:
  - 悬浮光晕: 宠物周围持续的柔和光圈
  - 冲锋轨迹: 从起点到终点的粒子拖尾
  - 命中特效: 根据宠物类型，有独立命中效果
```

### 3.4 状态效果视觉 (Status Effect Visuals)

#### 灼烧 (Burn) - 红墨沸腾

```
视觉:
  - 角色脚下产生红色墨池 (半径25px)
  - 墨池内气泡上升，破裂产生小火苗
  - 角色身体边缘泛红光
  
粒子参数:
  - 粒子数: 8-12个/秒
  - 寿命: 0.5-0.8秒
  - 颜色: #FF6B35 → #FF3D00 (渐变)
  - 大小: 3-6px
  - 形状: 圆形+星形混合
  - 运动: 垂直上升，轻微随机偏移
  
持续时间: 每次触发显示0.5秒，伤害时重新触发
```

#### 冰冻 (Freeze) - 蓝冰结晶

```
视觉:
  - 角色身体覆盖冰霜层 (alpha 0→0.6)
  - 冰晶从角色中心向外生长
  - 角色移动时碎裂冰片掉落
  
粒子参数:
  - 粒子数: 15-20个 (生成时一次性)
  - 寿命: 0.8-1.2秒
  - 颜色: #A2D2FF → #CAF0F8
  - 大小: 4-8px (六边形)
  - 形状: 六边形冰晶
  - 运动: 径向扩散，轻微旋转
  
特殊: 被冰冻时角色scale轻微缩小 (0.98)，移动速度-50%
```

#### 麻痹 (Paralyze) - 黄雷闪电

```
视觉:
  - 角色身体产生黄色电弧
  - 电弧随机跳跃，连接身体不同部位
  - 角色身体轻微抽搐 (位移±2px)
  
粒子参数:
  - 粒子数: 20-30个/秒 (断续)
  - 寿命: 0.1-0.2秒
  - 颜色: #FFE156 → #FFD700
  - 大小: 1-2px (线段)
  - 形状: 闪电分支
  - 运动: 随机路径，瞬间出现/消失
  
特殊: 有30%概率触发0.2秒无法行动
```

#### 中毒 (Poison) - 绿墨滴落

```
视觉:
  - 角色身体边缘滴落绿色墨滴
  - 墨滴落地产生小型毒池
  - 角色身体笼罩淡绿色雾气
  
粒子参数:
  - 粒子数: 5-8个/秒
  - 寿命: 1.0-1.5秒
  - 颜色: #2D6A4F → #52B788
  - 大小: 3-5px
  - 形状: 水滴形
  - 运动: 垂直下落，落地后水平扩散
  
特殊: 毒池累积，超过3层时爆发 (范围伤害)
```

#### 减速 (Slow) - 紫雾扩散

```
视觉:
  - 角色脚下产生紫色漩涡
  - 漩涡逆时针旋转，缓慢扩散
  - 角色身体拖出紫色残影
  
粒子参数:
  - 粒子数: 10-15个/秒
  - 寿命: 1.5-2.0秒
  - 颜色: #7B2D8E → #4A0E4E
  - 大小: 5-10px
  - 形状: 柔和圆形
  - 运动: 水平旋转，向外扩散
  
特殊: 移动速度-40%，动画播放速度-20%
```

### 3.5 战斗反馈 (Combat Feedback)

#### 3.5.1 命中闪烁 (Hit Flash)

```
普通命中:
  - 角色白色闪烁 (alpha 0.3→0)，持续0.1秒
  - 命中点产生小型墨溅

暴击命中:
  - 角色金色闪烁 (alpha 0.5→0)，持续0.2秒
  - 命中点产生大型墨溅 + 金色火花
  - 屏幕震动 (hitHeavy)
  
实现: shader uniform _FlashColor + _FlashIntensity，每帧衰减
```

#### 3.5.2 闪避拖尾 (Dodge Trail)

```
触发: 玩家成功闪避时
动画:
  - 玩家原位留下半透明残影 (alpha 0.6→0，0.3秒)
  - 残影边缘产生墨色粒子拖尾
  - 新位置出现时产生小型冲击波

参数:
```

### 6.2 性能优化策略

#### 6.2.1 粒子池管理

```
最大粒子总数: 500个 (同时存在)
分层优先级:
  - 高优先: 玩家技能、暴击效果 (200个配额)
  - 中优先: 敌人攻击、状态效果 (150个配额)
  - 低优先: 环境粒子、背景效果 (100个配额)
  - 备用: 50个

超限处理:
  1. 优先淘汰低优先粒子
  2. 同优先级淘汰最老粒子
  3. 新粒子替换最老粒子
```

#### 6.2.2 LOD (Level of Detail)

```
近距离 (<200px): 全特效
中距离 (200-500px): 粒子数-50%，禁用拖尾
远距离 (>500px): 粒子数-75%，禁用闪烁/旋转
屏幕外: 完全禁用

自动调整:
  - 帧率 <30fps: 全局粒子数-30%
  - 帧率 <20fps: 全局粒子数-50%
  - 帧率恢复: 逐步恢复粒子数
```

#### 6.2.3 批处理优化

```
相同材质的粒子合并渲染
相同混合模式的粒子分组
使用GPU Instancing减少Draw Call
粒子纹理图集: 4张512x512图集覆盖所有粒子形状
```
