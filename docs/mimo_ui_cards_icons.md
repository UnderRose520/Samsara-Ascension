## 1. 卡牌体系总览

### 1.1 四类卡牌定位

```
┌─────────────────────────────────────────────────────────────────────────┐
│                                                                         │
│  卡牌体系 = 游戏核心信息载体                                              │
│                                                                         │
│  ┌──────────────┐  ┌──────────────┐  ┌──────────────┐  ┌──────────────┐ │
│  │   词条卡      │  │   天赋卡      │  │   路径卡      │  │   道心卡      │ │
│  │  Affix Card   │  │ Talent Card   │  │  Path Card    │  │ DaoHeart Card │ │
│  │              │  │              │  │              │  │              │ │
│  │ 战斗中三选一  │  │ 破境后三选一  │  │ 战斗后选路径  │  │ 开局选难度    │ │
│  │ 构筑核心      │  │ 成长强化      │  │ 节奏控制      │  │ 道心叙事      │ │
│  │ 50+种词条    │  │ 每境3选1     │  │ 2-3条路径    │  │ 3种道心      │ │
│  │ 5种品质      │  │ 无品质区分    │  │ 风险/收益    │  │ 3种体验      │ │
│  └──────────────┘  └──────────────┘  └──────────────┘  └──────────────┘ │
│                                                                         │
│  共同DNA：暗色水墨底 + 宣纸纹理 + 金色/属性色强调 + 高饱和粒子点缀       │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

### 1.2 卡牌尺寸标准

| 卡牌类型 | 设计尺寸 (px) | 逻辑视口 (1280×720) | 1080p基准 | 纵横比 |
|----------|--------------|---------------------|-----------|--------|
| 词条卡 | 240×320 | 160×213 | 240×320 | 3:4 |
| 天赋卡 | 220×300 | 147×200 | 220×300 | ≈3:4 |
| 路径卡 | 280×200 | 187×133 | 280×200 | 7:5 |
| 道心卡 | 180×240 | 120×160 | 180×240 | 3:4 |

### 1.3 卡牌层级规则

```
卡牌在CanvasLayer中的归属：

  UI-04 词条三选一    → CanvasLayer 40 (Modal)
  UI-05 突破天赋三选一 → CanvasLayer 40 (Modal)
  UI-07 择路          → CanvasLayer 50 (Path/Shop)
  UI-01 道心Setup     → CanvasLayer 70 (Meta)

  词条卡/天赋卡 → Modal 层，战斗暂停时弹出
  路径卡 → Path 层，清场后战斗收尾时弹出
  道心卡 → Meta 层，全局遮罩下
```

## 2. 词条卡（Affix Card）

### 2.1 卡牌解剖

```
词条卡是游戏中最常见的卡牌类型。
玩家每次清完房间后，从3张词条卡中选择1张（散修4张）。

信息层级：
  1. 属性图标 — 一眼识别元素属性
  2. 词条名 — 大字标题，金色/品质色
  3. 品质标签 — 凡/灵/仙/天/道
  4. 分类标签 — 功法/法术/体质/神通/联动/伙伴
  5. 效果描述 — 2-3行说明
  6. 联动提示 — 可与哪些词条联动
  7. 选择按钮 — "择此机缘"
```

## 3. 天赋卡（Talent Card）

### 3.1 卡牌解剖

```
天赋卡在境界突破时出现，玩家从3张中选1张永久天赋。
与词条卡的区别：
  - 卷轴竖版造型（区别于词条卡的矩形）
  - 无品质区分（天赋均为固定效果）
  - 带境界图标（表示解锁的境界层级）
  - 无联动提示（天赋是独立被动）
  - 按钮文案"择此天赋"
```

## 5. 道心卡（Dao Heart Card）

```

只有3张：问道 / 悟道 / 证道
视觉上最庄重——代表"你以何种心念踏入仙途"。

道心卡是最小的卡牌（180×240），但信息密度最低。
每张卡只传递：道心名 + 副标题 + 核心差异 + 一句flavor。
```

## 6. 品质视觉体系

### 6.1 五阶品质视觉规范

```
品质是词条卡的核心视觉信号。
每阶品质有独立的边框、光效、粒子表现。
```

#### 凡品（Common）— 简墨

```
┌─────────────────────────┐
│  边框: 1px 实线 #B0B0B0  │  简单墨线勾勒，无装饰
│  光效: 无                 │  纯静态
```

### 7.2 各状态视觉规范

#### Default（默认态）

```
视觉表现：
  · 卡片位于原位，无偏移
  · 品质边框按品质等级显示（无扫光/脉冲）
  · 粒子系统低速运行（灵品及以上）
  · 按钮为默认金色

动画参数：
  · 无位移
  · 边框: 品质基础亮度
  · 粒子: 30% 速率
```

#### Hover（悬停态）

```
视觉表现：
  · 卡片上浮 translateY -4px
  · 边框亮度 +15%
  · 外发光半径 +4px
  · 按钮 hover 亮度 +8%
  · 其他两张卡轻微变暗（modulate @ 70%）

动画参数：
  · 位移: 120ms ease-out translateY -4px
  · 边框: 120ms ease brightness +15%
  · 光晕: 120ms ease radius +4px
  · 暗化: 120ms ease modulate 70%
```

#### Selected（选中态）

```
视觉表现：
  · 卡片放大 scale(1.05)
  · 边框全亮 + 最大外发光
  · 品质粒子全速运行
  · 按钮文案变"✓ 已选"
  · 其他卡灰显 modulate @ 40%
  · 卡片底部出现品质色光晕扩散

动画参数：
  · 缩放: 200ms spring scale(1.05)
  · 位移: 200ms ease translateY 0（回正）
  · 边框: 200ms ease 全亮
  · 粒子: 200ms ease 100% 速率
  · 灰显: 200ms ease modulate 40%
```

#### Rerolling（重随态）

```
视觉表现：
  · 所有卡牌同时执行重随动画
  · 卡片翻转180°（背面朝上）
  · 翻转过程中显示背面设计
  · 翻转完成后新卡从背面翻出
  · 灵石扣除动画（-50灵石飘字）

动画参数：
  · 翻转: 400ms ease-in-out rotateY(180°)
  · 停顿: 200ms（背面停留）
  · 翻回: 400ms ease-in-out rotateY(0°)
  · 新卡入场: stagger 100ms（从左到右）
  · 灵石飘字: 从卡区飞向左上灵石显示
```

#### Locked（锁定态 — 路径卡专属）

```
视觉表现：
  · 卡片灰显 modulate @ 30%
  · 边框消失，改为灰色虚线框
  · 图标变为灰色
  · 按钮禁用（disabled 样式）
  · 卡片上覆盖半透明灰色遮罩
  · 不可点击，hover 无反应

动画参数：
  · 无位移/缩放
  · 灰显: 300ms ease modulate 30%
```

## 10. 卡牌粒子特效

### 10.1 品质升级特效

```
当玩家选择词条卡时，根据品质触发不同粒子效果：

  凡品 → 选择
    · 无粒子效果
    · 卡片淡出 + 向左上飞入HUD

  灵品 → 选择
    · 蓝色萤光从卡片中心向外扩散
    · 3-5颗蓝色粒子飘散
    · 卡片边框蓝色闪光一次
    · 淡出飞入HUD

  仙品 → 选择
    · 紫色星点从卡片中心爆发
    · 8-12颗紫色粒子环绕卡片旋转一圈
    · 卡片边框紫色脉冲2次
    · 屏幕边缘微弱紫色光晕
    · 淡出飞入HUD

  天品 → 选择
    · 金色星尘从上方飘落到卡片
    · 15-20颗金色粒子从卡片爆发
    · 卡片边框金色扫光
    · 屏幕边缘金色光晕扩散
    · "领悟！" Toast弹出
    · 淡出飞入HUD

  道品 → 选择
    · 红金双色粒子从卡片中心旋涡状爆发
    · 20-30颗红金粒子向四周扩散
    · 全屏短暂闪白0.1秒
    · 卡片边框红金脉冲3次
    · "道之极致！" Toast弹出
    · 屏幕边缘红金光晕
    · 淡出飞入HUD
```

### 10.2 卡牌揭示特效

```
三张卡牌从背面翻出的揭示动画：

  阶段1: 翻转揭示 (400ms)
    · 三张卡片同时从背面翻转到正面
    · 翻转过程中：背面→侧面→正面
    · 背面朝上时：只有轮回符号
    · 翻转完成：品质色光芒从卡片中心闪现

  阶段2: 品质确认 (200ms)
    · 翻转完成后，品质粒子开始初始化
    · 灵品以上：边框光效淡入
    · 仙品以上：粒子系统启动

  阶段3: 入场完成 (100ms)
    · 卡片位置稳定
    · hover检测开始
    · 玩家可操作

  整体效果：像从牌堆中翻出三张牌
  翻转轴: Y轴（左右翻转，不是上下）
  透视: perspective(800px)
```

### 10.3 选择发光特效

```
当玩家hover/选择卡牌时的光效：

  Hover光效:
    · 卡片底部出现向上扩散的属性色光晕
    · 光晕高度: 卡高的30%
    · 光晕透明度: 属性色@15% → @25%
    · 动画: 120ms ease-in-out
    · 其他卡: modulate @ 70%

  选择光效:
    · 卡片底部光晕扩大到卡高的50%
    · 光晕透明度: @25% → @40%
    · 光晕向上扩散到整个卡面
    · 卡片中心出现属性色光点
    · 其他卡: modulate @ 40%
    · 动画: 200ms spring
```

### 10.4 元素粒子系统

```
每种元素有专属的粒子表现：

  火系粒子:
    · 形状: 小火焰/火星
    · 运动: 向上飘动 + 随机横向摆动
    · 颜色: #FF6B35 → #FFD700 渐变
    · 寿命: 0.5-1秒
    · 大小: 2-4px
    · 适用: 火系词条卡、火系法术图标

  水系粒子:
    · 形状: 水滴/涟漪
    · 运动: 从中心向外扩散的同心圆
    · 颜色: #4ECDC4 → #88D4D0 渐变
    · 寿命: 0.8-1.2秒
    · 大小: 2-6px (涟漪)
    · 适用: 水系词条卡、水系法术图标

  雷系粒子:
    · 形状: 电弧/闪电碎片
    · 运动: 从中心向外跳跃式扩散
    · 颜色: #FFD700 → #FFFFFF 渐变
    · 寿命: 0.2-0.5秒 (快速闪烁)
    · 大小: 1-3px
    · 适用: 雷系词条卡、雷系法术图标

  木系粒子:
    · 形状: 叶片/花粉
    · 运动: 从下向上缓慢飘动
    · 颜色: #7BC67E → #A8E6A8 渐变
    · 寿命: 1-2秒
    · 大小: 2-5px
    · 适用: 木系词条卡、木系法术图标

  土系粒子:
    · 形状: 碎石/尘埃
    · 运动: 从中心向四周溅射
    · 颜色: #C4A35A → #E0C882 渐变
    · 寿命: 0.5-1秒
    · 大小: 1-4px
    · 适用: 土系词条卡、土系法术图标

  混沌粒子:
    · 形状: 旋涡/星点
    · 运动: 螺旋旋转
    · 颜色: #B57EDC → #D4A5F7 渐变
    · 寿命: 0.8-1.5秒

    · 适用: 混沌系词条卡、五行归元
```
