## 设计总纲

### 核心美学

本方案以 **中国传统水墨画** 为视觉根基，融合 **现代暗色游戏 UI** 的可读性与 **高饱和粒子特效** 的视觉冲击力。整体画面如同一幅在暗夜中缓缓展开的水墨长卷——浓墨为底，金色铭文浮现，粒子如萤火般跃动。

### 五大视觉原则

| # | 原则 | 释义 |
|---|------|------|
| 1 | **墨分五色** | 仅用墨色浓淡变化构建80%的画面层次，色彩只在关键信息处爆发 |
| 2 | **计白当黑** | 留白即设计，中心60%战斗区域永远保持最大净空 |
| 3 | **气韵生动** | 所有静态元素都有微弱的呼吸动画，UI是活的 |
| 4 | **暗底跳亮** | 粒子饱和度90-100%从暗底跳出，绝不与背景融合 |
| 5 | **仪式感至上** | 突破、觉醒、道消等时刻是非跳过的视觉盛宴 |

### 统一色彩体系（修正后）

## 动画与特效体系概览

### 水墨缓动曲线

| 曲线 | 数学描述 | 用途 | 视觉感受 |
|------|---------|------|---------|
| `inkEaseIn` | `cubic-bezier(0.6, 0.0, 1.0, 0.4)` | 墨色渗入 | 墨滴入水，凝聚后急散 |
| `inkEaseOut` | `cubic-bezier(0.0, 0.6, 0.4, 1.0)` | 效果消散 | 毛笔离纸，先重后轻 |
| `inkEaseInOut` | `cubic-bezier(0.7, 0.1, 0.3, 0.9)` | 场景转场 | 完整笔锋起落 |
| `brushStroke` | `cubic-bezier(0.2, 0.8, 0.3, 1.0)` | 技能释放 | 笔触从粗到细 |
| `sealStamp` | `cubic-bezier(0.8, 0.0, 0.2, 1.0)` | 暴击触发 | 突然冲击后沉降 |

### 关键时刻演出时长

| 时刻 | 时长 | 核心效果 |
|------|------|---------|
| 天机一击（暴击） | 1.5s | 时间冻结 + 金色铭文爆炸 + 慢动作 |
| 境界突破 | 3-5s | 墨色溶解 + 境界符号绘制 + 力量波 |
| 道统觉醒 | 4-6s | 全屏暗化 + 书法自绘 + 金光爆发 |
| 道消（死亡） | 5-8s | 时间冻结 + 去色 + 墨化消散 + 铭文 |
