## 全局面板视觉规则

### 面板共通结构

所有非战斗面板共享三层视觉结构：

```
┌─────────────────────────────────────────────────────────────────┐
│  [第一层] 全屏遮罩                                               │
│  bg.deep #0D0D0D @ 85-95% 不透明度                              │
│  可选：极淡宣纸纹理叠加 @ 4% 透明度                               │
│  可选：底部灵气粒子流（慢速、低密度、半透明）                       │
│                                                                 │
│  [第二层] 面板主体                                               │
│  bg.panel #1A1A2E 底 + 内描边 1px #FFFFFF @ 8%                  │
│  四角卷轴角饰（NinePatch 金色 L 形金属线，非全框）                 │
│  顶部标题区 + 内容区 + 底部操作区                                  │
│                                                                 │
│  [第三层] 焦点元素                                               │
│  高亮卡片 / 按钮 / 粒子特效 / 文字描边                            │
│  使用属性色或品质色发光                                           │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 面板入场动画（通用）

```
t=0ms     遮罩 opacity 0% → 开始淡入
t=0-180ms 遮罩 opacity 0% → 85% (ease-in)
t=100ms   面板主体 scale 0.96 → 1.0 + opacity 0% → 100%
t=100-350ms 面板缩放+淡入 (ease-out, 250ms)
t=350ms   焦点元素入场延迟动画开始（卡片逐个滑入、粒子爆发等）
总耗时约 350ms，玩家感知为"一瞬间展开"
```

### 面板关闭动画（通用）

```
t=0ms     所有元素 opacity 100% → 开始淡出
t=0-180ms 面板 + 遮罩 opacity → 0% (ease-in)
t=180ms   面板移除
总耗时 180ms，快速不拖沓
```

  三张卡片同时 rotateY 90° (200ms)
  新卡片 rotateY -90° → 0° (200ms)
  伴随灵石扣除数字飘出

跳过动画：
  面板淡出 + 灵石+30 数字从中央飘向右上角

### 动画——突破仪式（核心高光）

```
阶段一：聚势（0-800ms）
  角色原地站立，周围灵气粒子加速向角色聚拢
  粒子密度从低到高，速度从慢到快
  背景遮罩从 55% → 70% 加深

阶段二：爆发（800-1200ms）
  t=800ms  角色身上属性色光环急速扩大
  t=900ms  灵气从角色位置向屏幕四周爆发
  t=1000ms 全屏白闪 0.15秒
  t=1150ms 白闪消退，画面恢复

阶段三：展示（1200ms+）
  境界名字从中央放大弹出（scale 0.5→1.0 + 淡入，300ms）
  词条槽数字变化动画
  天赋卡片从下方依次滑入

音效：
  聚势阶段：低频嗡鸣渐强
  爆发阶段：庄严钟鸣 + 属性专属音效
  展示阶段：空灵回响渐弱

整个仪式不可跳过（核心成就体验）
但可加速（设置中"减少动效"→ 仪式缩短至1秒）
```

### 面板场景结构建议

```
每个面板 .tscn 的根节点：
  CanvasLayer（设置对应层级）
    └─ ColorRect（全屏遮罩，mouse_filter=STOP）
        └─ PanelContainer（面板主体，anchored 居中）
            └─ MarginContainer（内边距 space.xl=24px）
                └─ VBoxContainer（主布局）
                    ├─ HBoxContainer（标题区）
                    ├─ Control/ScrollContainer（内容区）
                    └─ HBoxContainer（操作区）
```

### 信号对齐

```
面板通用信号：
  panel_opened     → 暂停战斗逻辑
  panel_closed     → 恢复战斗逻辑
  choice_made(data) → 返回玩家选择

与 EventBus 对齐：
  affix_choice_requested    → 打开词条三选一
  breakthrough_triggered    → 打开境界突破
  path_choice_requested     → 打开择路
  shop_entered              → 打开坊市
  event_triggered(event_id) → 打开随机事件
  dao_tradition_awakened    → 播放道统觉醒
  player_died               → 播放道消 → 结算
  run_ended                 → 打开结算
```
