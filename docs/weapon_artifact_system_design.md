# 《轮回仙途》武器 / 法宝 / 器灵系统设计案

> 结论：应该加入武器，而且玩家和怪物都应该有自己的武器。但不能做成普通刷装系统。武器在本项目里应该是“身份、攻击形态、战斗读招、器灵觉醒”的支架。

## 1. 为什么需要武器

当前玩法如果只有“功法词条 + 远程法术”，会出现几个问题：

- 玩家身份不够清晰：剑修、法修、符修、邪修的差异只停留在名字。
- 攻击形态缺根：为什么这个角色近战？为什么那个角色远程？为什么能召魂？
- 怪物识别弱：敌人如果只按 archetype 区分，玩家很难一眼知道它要做什么。
- 器灵缺载体：器灵如果没有具体器物，就会像额外技能，不像“器中有灵”。
- 掉落来源不自然：残卷、法宝、材料、灵石需要更可信的来源。

武器可以解决这些问题：

```text
道途决定本命器
本命器决定普攻形态
功法改造本命器
法宝提供副能力
器灵从器物中觉醒
怪物武器提供读招信息
```

## 2. 最大风险

武器系统很诱人，但风险也很大。

不要做成：

```text
黑铁剑 +3
青铜剑 +5
白银剑 +7
黄金剑 +10
```

这会把游戏拖向刷装备，而不是修仙悟道。玩家会盯数值，不再关心道途、天象、悟道和轮回。

本项目的武器应该遵守三条：

1. **武器先定义玩法，再定义数值。**
2. **本命器少换，重在祭炼成长。**
3. **法宝可获得，但数量少、效果强、来源明确。**

## 3. 概念拆分

需要把“武器、法器、法宝、器灵”拆清楚。

| 概念 | 定义 | 游戏作用 |
|---|---|---|
| 本命器 | 开局随道途绑定的主武器/施法媒介 | 决定普攻形态和基础战斗距离 |
| 功法 | 修炼法门 | 改造本命器和法术 |
| 法术 | 主动招式 | Q/E/R 等技能 |
| 法器 | 低阶可用器物 | 局内早期副装备或敌人武器 |
| 法宝 | 成熟的强力器物 | 稀有副装备，带被动/主动 |
| 器灵 | 器物觉醒的人格 | F 技能、对话、成长线 |

一句话：

```text
本命器是你的手，功法是你的道，法宝是你的外物，器灵是外物生出的伙伴。
```

## 4. 武器分类如何参考

用户给出的分类非常丰富，可以参考，但必须收束。不要一次性把剑、刀、枪、弓、锤、斧、针、环、钉、琴、盾、伞、幡、印、绳、索、珠、图、斗、扇、旗、钟、鼎、塔、壶、瓶、葫、镜全做成独立系统。

建议按“玩法形态”合并成 8 类。

| 大类 | 包含形态 | 玩法定位 | 适合谁 |
|---|---|---|---|
| 剑刃 | 剑、刀、飞剑 | 近中距离、斩线、连段 | 剑修、散修、剑魂敌 |
| 长兵 | 枪、戟、矛 | 直线突刺、冲锋、破甲 | 体修敌、守关将 |
| 远射 | 弓、箭、针、钉 | 远程狙击、穿透、标记 | 弓修敌、机关傀儡 |
| 重器 | 锤、斧、印、钟、鼎、塔 | 震地、压制、护盾、范围控制 | 体修、Boss、机关敌 |
| 法媒 | 珠、扇、镜、葫、瓶、壶 | 施法媒介、属性法术、吸纳/释放 | 法修、丹修 |
| 阵符 | 旗、图、斗、符笔 | 布阵、延迟触发、区域控制 | 符修、阵法师 |
| 缚控 | 环、绳、索、伞 | 牵引、束缚、防御、位移 | 控制型敌、辅助法宝 |
| 魂邪 | 幡、铃、骨器、血器 | 召魂、诅咒、献祭、心魔 | 邪修、魔修、心魔 |

这样做的好处：

- 玩家看到敌人拿弓，就知道是远程威胁。
- 玩家看到魂幡，就知道会召唤/诅咒。
- 开发上只需要做 8 套行为框架，而不是 30 套。

## 5. 品阶与材质如何使用

用户提供的“黑铁、青铜、白银、黄金、耀金、紫晶、星辰、耀月、炽阳、创世/毁灭、神兵”等更像泛奇幻材质。可以参考，但建议修仙化，并减少层级。

### 推荐品阶

主线一局 15-30 分钟，不适合放太多装备等级。建议局内只显示 5 档：

| 品阶 | 定位 | 出现阶段 |
|---|---|---|
| 凡器 | 普通器胚 | 开局/低阶敌人 |
| 法器 | 初步炼化 | 炼气/筑基 |
| 灵器 | 有灵性 | 筑基/金丹 |
| 法宝 | 成熟强器 | 金丹/元婴 |
| 灵宝 | 稀有机缘 | 元婴/渡劫/隐藏 |

仙器、神器、道器可以作为长期/结局/无尽内容，不要早期泛滥。

### 推荐子品

如果需要下品、中品、上品、极品，建议只作为细分，不放到所有 UI 主标题里。

示例：

```text
灵器 · 上品 · 雷纹剑
法宝 · 中品 · 镇魂幡
```

不要让玩家每分钟比较“中品白银剑”和“上品青铜剑”。这会很累。

### 材质作为词缀，不作为主等级

材质适合做“来源和特色”：

| 材质 | 玩法倾向 |
|---|---|
| 玄铁 | 破甲、剑刃、重器 |
| 赤铜 | 火、爆燃、丹炉 |
| 寒玉 | 水冰、护盾、冻结 |
| 雷击木 | 雷、导电、符箓 |
| 星砂 | 暴击、飞剑、镜 |
| 月华银 | 冷却、身法、幻影 |
| 血骨 | 邪修、献祭、吸血 |
| 魂晶 | 召魂、心魔、诅咒 |

最终显示可以是：

```text
雷击木 · 引雷符旗
星砂 · 青锋飞剑
魂晶 · 镇魂幡
```

材质帮助叙事和构筑，不制造无意义等级膨胀。

## 6. 属性分类如何使用

属性不要一次性铺太多。首发建议：

| 属性 | 用途 |
|---|---|
| 火 | 爆燃、持续伤害、传播 |
| 水/冰 | 减速、冻结、护盾 |
| 雷 | 麻痹、导电、斩首 |
| 木/毒 | 毒、召唤、回复 |
| 土/金 | 护盾、震地、破甲 |
| 风 | 位移、弹道、扩散 |
| 魂 | 召魂、诅咒、心魔 |
| 血 | 献祭、吸血、燃命 |
| 空间 | 位移、镜像、传送 |
| 时间 | 冷却、延缓、回溯 |

首发只做火、雷、水冰、魂四类就足够。

## 6.1 品质显示层级

要避免三套品质系统同时抢注意力：

```text
词条品质：凡 / 灵 / 仙 / 天 / 道
武器品阶：凡器 / 法器 / 灵器 / 法宝 / 灵宝
材料品质：玄铁 / 寒玉 / 星砂 / 魂晶……
```

建议 UI 上遵守：

- 战斗奖励卡主标题优先显示“玩法效果”，不是品阶。
- 本命器界面显示武器品阶。
- 道谱界面显示词条/功法品质。
- 材料只在祭炼/炼化界面显示。

示例：

```text
错误：灵器·上品·星砂·仙品·青锋剑·火借风势
正确：青锋剑铭纹：火借风势
副信息：灵器上品 / 星砂铭纹 / 来源：烈阳裂地
```

玩家先看懂玩法，再看稀有度。

## 7. 玩家本命器设计

本命器是开局道途的核心。它不应该频繁替换，而应该在一局中被功法、材料、天象和突破不断祭炼。

### 首发道途与本命器

| 道途 | 本命器 | 普攻形态 | 开局 Q | 后续成长 |
|---|---|---|---|---|
| 剑修 | 青锋剑 / 飞剑 | 三段近战斩，第三段短剑芒 | 御剑突刺 | 剑芒、回身斩、飞剑驻留、剑魂 |
| 法修 | 法珠 / 法扇 / 法印 | 远程元气弹 | 属性弹三选一 | 改属性、蓄法、法阵、法宝共鸣 |
| 符修 | 符笔 / 符旗 | 投符，落地成符 | 起符 | 连符、三符成阵、天象改符 |
| 体修 | 拳套 / 臂甲 / 重锤 | 近战拳掌/震地 | 崩山劲 | 霸体、血气、反震、金身 |
| 丹修 | 丹炉 / 葫芦 | 药气波/毒丸 | 炼丹火 | 回血、毒雾、丹爆、炉魂 |
| 邪修 | 魂幡 / 血铃 | 魂火/血刃 | 召魂幡影 | 召魂、献祭、诅咒、心魔交易 |

这解决了“为什么都是远程”的问题：只有法修天然远程，其他道途各有距离和节奏。

## 8. 武器不是随机掉落，而是祭炼成长

建议本命器有 4 层成长，不做频繁换装。

| 层 | 名称 | 触发 | 效果 |
|---|---|---|---|
| 1 | 器胚 | 开局自带 | 决定普攻形态 |
| 2 | 淬炼 | 关卡材料/清场灵机 | 小幅改变攻击范围/状态 |
| 3 | 铭纹 | 词条/道谱槽位 | 接入属性和天象 |
| 4 | 通灵 | 特定行为/器灵线索 | 解锁器灵觉醒或 F 技能 |

例子：剑修青锋剑。

```text
器胚：三段剑斩
淬炼：第三段剑芒更长
铭纹：剑芒可附火/雷/冰
通灵：剑魂感应，完美闪避后飞剑自动护主
```

例子：邪修魂幡。

```text
器胚：魂火弹 + 小魂影
淬炼：击杀留残魂
铭纹：残魂可爆/可缚/可献祭
通灵：幡灵低语，F 释放百魂夜行
```

## 9. 法宝作为副装备

法宝可以存在，但要少而强。建议一局最多 2-3 件，每件有明确用途。

| 法宝 | 形态 | 定位 |
|---|---|---|
| 玄铁飞剑 | 飞剑 | 自动斩击、剑魂觉醒 |
| 天雷珠 | 珠 | 落雷、导电、雷灵觉醒 |
| 焚天炉 | 鼎/炉 | 火场、炼化、炉魂觉醒 |
| 玄武甲 | 甲/盾 | 护盾、反伤、龟灵觉醒 |
| 探宝罗盘 | 盘/镜 | 隐藏房、掉落、鉴灵觉醒 |
| 镇魂幡 | 幡 | 召魂、诅咒、幡灵觉醒 |
| 缚妖索 | 索 | 牵引、束缚、控场 |
| 时空镜 | 镜 | 短回溯、镜像、镜灵觉醒 |

### 法宝和本命器的区别

| 项 | 本命器 | 法宝 |
|---|---|---|
| 来源 | 开局道途 | 局内机缘/坊市/Boss |
| 数量 | 1 个 | 1-3 个 |
| 是否常换 | 少换，重成长 | 可替换/购买 |
| 作用 | 普攻和身份 | 被动、主动、器灵 |
| 觉醒 | 可有本命器灵 | 更常见器灵 |

## 10. 器灵从武器中生长

器灵不应是随机抽到的伙伴，而应从器物行为中被“养醒”。

| 器物 | 觉醒条件 | 器灵 | F 技能 |
|---|---|---|---|
| 玄铁飞剑 | 飞剑命中/完美闪避后斩杀 | 剑魂 | 万剑归宗 |
| 天雷珠 | 雷暴天导电击杀/落雷命中 | 雷灵 | 引雷入体 |
| 焚天炉 | 爆燃击杀/炼化妖丹 | 炉魂 | 天地熔炉 |
| 玄武甲 | 护盾吸收/濒死不倒 | 龟灵 | 玄武结界 |
| 探宝罗盘 | 发现隐藏/选择机缘 | 鉴灵 | 鉴宝天眼 |
| 镇魂幡 | 召魂击杀/恶业因果 | 幡灵 | 百魂夜行 |
| 时空镜 | 完美闪避/濒死回避 | 镜灵 | 回溯三息 |

这能把玩家行为和器灵觉醒绑定起来，避免“器灵凭空出现”。

## 11. 怪物也应该有武器

怪物拥有武器非常重要，因为它能解决读招问题。

玩家在肉鸽动作游戏里需要快速判断威胁。武器轮廓是最直观的提示。

### 敌人武器与行为

| 敌人武器 | 玩家读到的信息 | 行为 |
|---|---|---|
| 爪/短刀 | 会贴脸、攻速快 | 追击、连抓 |
| 长枪 | 会直线突刺 | 预警线、冲锋 |
| 弓/弩 | 远程高伤 | 狙击、穿透箭 |
| 扇 | 扇形弹幕 | 多发散射 |
| 盾/甲 | 防御/护阵 | 护盾、挡弹 |
| 锤/印 | 震地 | 范围预警、击退 |
| 符旗 | 布阵 | 地面阵、强化怪群 |
| 魂幡 | 召唤/诅咒 | 召魂、减速、心魔 |
| 镜 | 分身/反射 | 镜像、反弹 |
| 鼎/炉 | 火/毒/炼化 | 火场、毒雾、爆炸 |

### 敌人武器与掉落

敌人武器也解释奖励来源：

| 击败对象 | 合理掉落 |
|---|---|
| 弓修 | 穿云残卷、弓弦、风系机缘 |
| 符师 | 符纸、符笔、阵图 |
| 魂幡邪修 | 魂晶、邪法残页、恶业因果 |
| 机关傀儡 | 机关核、玄铁、法宝碎片 |
| 鼎炉魔修 | 炉灰、丹毒、焚天炉线索 |

这样奖励不会像系统发牌，而像从敌人身上来。

## 12. 玩家与怪物武器的对称性

最好的设计是：玩家看到怪物会用某类武器，也会幻想自己能得到相近路线。

例如：

```text
你被符师的连环符阵打得很惨。
击败后获得《三才符解》残页。
下一局你可以选择符修道途。
```

或：

```text
魂幡邪修召出残魂围攻你。
你击败他后得到镇魂幡碎片。
若选择炼化，会获得魂系机缘；若选择镇压，会获得道心因果。
```

这样敌人不只是障碍，也是教学和诱惑。

## 13. 武器与道谱

道统条件应该允许“武器槽”参与。

例子：

### 剑火道脉

| 槽位 | 条件 |
|---|---|
| 本命器 | 剑类 |
| 火源 | 小离火诀 / 火焰弹 |
| 传播 | 火借风势 / 剑芒附火 |
| 行为 | 剑芒爆燃击杀 20 |

### 镇魂道统

| 槽位 | 条件 |
|---|---|
| 器物 | 魂幡 / 血铃 / 魂晶 |
| 召魂 | 残魂召唤类 |
| 代价 | 恶业或献祭 |
| 行为 | 召魂击杀精英 3 次 |

### 炉火丹道

| 槽位 | 条件 |
|---|---|
| 器物 | 丹炉 / 葫芦 |
| 火源 | 炉火 / 焚天炉 |
| 炼化 | 妖丹炼化 |
| 行为 | 炼化材料 5 次或丹爆击杀 |

武器让道统不再只是属性集合，而是“你以什么器、用什么法、做过什么事”。

## 14. 首批武器建议

不要从 30 种开始。首批 8 个足够撑起差异。

| 武器/法宝 | 使用者 | 首发理由 |
|---|---|---|
| 青锋剑 | 剑修 / 剑修敌 | 解决近战身份 |
| 法珠 | 法修 | 保留远程施法 |
| 符旗/符笔 | 符修 / 符师敌 | 做阵地玩法 |
| 丹炉/葫芦 | 丹修 / 炉系敌 | 炼化和火毒 |
| 魂幡 | 邪修 / 魔修敌 | 召魂、心魔、因果 |
| 天雷珠 | 雷法宝 | 天象联动强 |
| 玄武甲/盾 | 防御法宝/盾敌 | 护盾和反击 |
| 镜 | 心魔/Boss/法宝 | 分身、回溯、反射 |

第二批再扩：

```text
枪、弓、扇、索、钟、塔、琴、环
```

## 15. 武器不该怎么掉

不建议：

```text
每个房间掉 3 把武器
比较 DPS
换上更高品质
卖掉旧武器
```

建议：

```text
本命器不频繁换
局内获得材料/铭纹/法宝碎片
通过事件或 Boss 祭炼
少量法宝作为强力副装备
器灵靠行为觉醒
```

奖励类型：

| 奖励 | 作用 |
|---|---|
| 器胚 | 解锁新道途/新本命器 |
| 材料 | 淬炼本命器 |
| 铭纹 | 给武器接入属性/天象 |
| 法宝碎片 | 合成副法宝 |
| 器灵线索 | 加速觉醒 |
| 残卷 | 解锁武器相关功法 |

## 16. 最小实现方案

### P0：只做身份，不做掉落

目标：让玩家和怪物看起来已经有武器。

内容：

- 开局道途绑定 `weapon_id`。
- 玩家普攻根据 `weapon_family` 改变。
- 敌人数据增加 `weapon_id`。
- HUD/开局卡显示本命器。

示例：

```csv
path_id,name,weapon_id,basic_attack_type,start_q
sword,剑修,qing_feng_sword,melee_combo,yu_jian_thrust
caster,法修,starter_orb,projectile,lie_yan_bolt
talisman,符修,san_cai_flag,talisman_trap,qi_fu
```

### P1：怪物武器读招

目标：玩家看到武器就知道怎么躲。


## 18. 最终建议

应该加入武器，而且要尽早加入“本命器”概念。

但第一阶段只做：

```text
剑修：青锋剑
法修：法珠
符修：符旗/符笔
邪修敌：魂幡
雷系法宝：天雷珠
```

先用它们解决三个核心问题：

1. 玩家身份不清。
2. 开局全远程。
3. 敌人行为不好读。

等这三件事成立，再扩材料、品阶、法宝碎片、器灵觉醒。

## 19. 不能牺牲肉鸽随机性

本命器不能把游戏做死。它负责“我是谁”，但不能规定“这一局只能这么玩”。

正确方向：

```text
开局本命器固定身份
局中铭纹随机改造
法宝随机改变节奏
器灵随机形成高光
天象随机推动变异
```

也就是说，剑修每局都有剑，但每局不该都是同一把剑的同一种打法。

### 本命器固定，祭炼随机

本命器的基础形态固定，但每层祭炼都应该给随机分支。

例子：青锋剑。

| 祭炼层 | 随机分支 | 玩法变化 |
|---|---|---|
| 淬炼 | 回风 | 闪避后反斩 |
| 淬炼 | 破竹 | 第三段剑斩穿透 |
| 淬炼 | 藏锋 | 停手蓄势，下一剑暴击 |
| 铭纹 | 雷击木 | 剑芒导电 |
| 铭纹 | 寒玉 | 剑芒减速/冻结 |
| 铭纹 | 血骨 | 击杀回血但增加心魔 |
| 通灵 | 剑魂护主 | 完美闪避自动飞剑 |
| 通灵 | 剑魂争鸣 | 连斩越多剑芒越密 |

这样玩家不是“选择剑修后就知道全局答案”，而是每局在不同材料、天象和事件中把剑炼歪。

### 武器随机池的三种奖励

武器相关奖励也要遵守肉鸽三段式：

| 奖励类型 | 目的 | 示例 |
|---|---|---|
| 稳定增强 | 让当前打法更顺 | 剑斩范围 +10%，法珠弹速 +15% |
| 构筑推进 | 帮玩家接近流派质变 | 雷击木铭纹、三符成阵、魂幡摄魂 |
| 破格诱惑 | 强行打开新路线 | 引劫入剑、血祭开锋、镜中第二本命器 |

破格诱惑尤其重要。它让玩家产生“这局能不能赌一把”的肉鸽欲望。

### 武器与法宝的随机组合示例

| 本命器 | 随机法宝 | 形成玩法 |
|---|---|---|
| 青锋剑 | 天雷珠 | 雷剑导电清屏 |
| 青锋剑 | 时空镜 | 完美闪避回溯剑 |
| 青锋剑 | 镇魂幡 | 击杀养魂，剑斩带诅咒 |
| 法珠 | 焚天炉 | 火法炼化爆燃 |
| 法珠 | 缚妖索 | 拉怪后范围法术清场 |
| 符旗 | 风雷扇 | 符阵扩散，雷符连锁 |
| 丹炉 | 魂晶 | 毒雾杀敌生魂 |
| 魂幡 | 玄武甲 | 召魂抗线，诅咒消耗 |

这些组合要有点“意外但合理”。玩家看到组合时应该想：原来还能这么修。

### 怪物武器也要随机变体

怪物持武器不是固定模板，也可以有变体。这样同一敌人不会每次都一样。

| 基础敌人 | 武器变体 | 行为变化 |
|---|---|---|
| 散修剑客 | 雷纹剑 | 突刺后留下导电剑痕 |
| 散修剑客 | 血锈剑 | 命中造成流血，低血狂暴 |
| 魂幡邪修 | 破魂幡 | 少量召魂，高频诅咒 |
| 魂幡邪修 | 百鬼幡 | 大量召唤，施法慢 |
| 符师 | 火符旗 | 地面爆燃阵 |
| 符师 | 风符旗 | 推拉、扩散弹道 |
| 弩修 | 穿云弩 | 长预警高伤 |
| 弩修 | 毒针匣 | 短预警多段毒针 |

玩家仍然能通过轮廓读懂大方向，但每局会遇到不同压力。

### 每局至少一个武器惊喜

建议每局强制保底出现 1 个武器相关惊喜事件：

| 事件 | 效果 |
|---|---|
| 器冢裂隙 | 三选一本命器铭纹，可跨属性 |
| 妖骨炼器 | 消耗生命上限，立刻淬炼本命器 |
| 雷火淬锋 | 当前天象属性注入本命器 |
| 残魂认主 | 若本局击杀足够多，提前获得器灵线索 |
| 夺器之战 | 精英敌持有法宝，击败后获得碎片 |
| 镜中异器 | 获得一个临时副本命器，持续一房 |

这些事件会把武器系统从“装备栏”变成肉鸽核心惊喜。

### 武器爽点检查

新增武器、法宝或铭纹前，必须问：

1. 它有没有改变攻击形状，而不只是加数值？
2. 它能不能和至少 2 种天象/属性产生组合？
3. 它有没有一个清屏、爆发、连锁、反杀或召唤类爽点？
4. 它会不会让同一道途的一局和下一局明显不同？
5. 它的来源是否能被敌人、事件、Boss 或前世解释？

如果只满足“攻击 +10%”，就不值得作为武器奖励出现。
