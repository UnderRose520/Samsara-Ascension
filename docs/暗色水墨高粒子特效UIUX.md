# 《轮回仙途》场景地图总纲 + 各天气详细描述

> 核心画风锚定：**暗色水墨底 × 高饱和粒子特效**
> 所有天气场景共享同一套底层视觉规则，区别只在色调、地形和粒子类型。

---

## 一、画风总纲——"暗色水墨 × 高饱和粒子"

### 三层视觉结构

```
┌─────────────────────────────────────────────────────┐
│                                                     │
│  第三层（最上层）：高饱和粒子特效                        │
│  ┌───────────────────────────────────────────────┐  │
│  │ 技能命中、连锁爆炸、万法归一、道势满值            │  │
│  │ 属性色粒子（火橙/雷蓝/冰银/毒绿/剑金）          │  │
│  │ 飘字、震屏、全屏闪光                             │  │
│  │ 是画面中色彩最浓烈的部分                          │  │
│  └───────────────────────────────────────────────┘  │
│                                                     │
│  第二层（中层）：天象粒子 + 地形光效                    │
│  ┌───────────────────────────────────────────────┐  │
│  │ 雨丝、雪花、沙尘、热浪、灵光                      │  │
│  │ 积水反光、裂地红光、冰面银光、毒雾绿光            │  │
│  │ 半透明覆盖层，不完全遮挡底层                       │  │
│  │ 颜色饱和度中等，比底层亮但比技能特效弱             │  │
│  └───────────────────────────────────────────────┘  │
│                                                     │
│  第一层（底层）：暗色水墨场景                          │
│  ┌───────────────────────────────────────────────┐  │
│  │ 深灰近黑的地面，带纸纹/石纹水墨质感              │  │
│  │ 浓墨轮廓的障碍物，不规则边缘                      │  │
│  │ 深靛/深紫/深红的天空渐变                          │  │
│  │ 画面边缘墨色晕染，像宣纸自然洇开                  │  │
│  │ 饱和度极低，几乎只有明暗关系                       │  │
│  └───────────────────────────────────────────────┘  │
│                                                     │
└─────────────────────────────────────────────────────┘
```

### 底层水墨质感——所有场景通用

```
地面纹理：
  - 底色：深灰近黑（RGB约 30-45, 28-40, 35-50）
  - 纹理：极淡的水墨笔触痕迹，像在生宣上用淡墨扫过
  - 石板接缝：用更浓的墨线勾勒，线条不规则、有毛边
  - 苔藓/裂痕：用湿墨晕染表示，边缘模糊自然过渡

障碍物（石头、残柱、断墙）：
  - 纯墨色剪影，外轮廓用浓墨线勾勒
  - 内部有淡墨渐变表示体积感
  - 不画细节纹理——只靠明暗和轮廓
  - 边缘有墨色飞白效果（像毛笔快速扫过留下的飞白）

天空/背景：
  - 从画面顶部到底部的渐变，顶部最深，靠近地平线略亮
  - 渐变不是均匀的，有墨色团块的不规则变化
  - 天空本身没有星星、云朵等细节——只有明暗渐变

画面边缘：
  - 四周有墨色晕染渐变，像画面边缘自然洇开
  - 角落最深，向内逐渐变淡
  - 这个效果替代了传统的黑边/vignette，但更有水墨气质

整体饱和度：
  - 底层场景的饱和度极低（近乎灰度）
  - 只有明暗关系，没有鲜艳色彩
  - 所有鲜艳色彩来自第二层（天象）和第三层（技能）
```

### 第二层天象粒子——按天气变化

```
这层是"暗色水墨底"和"高饱和粒子"之间的过渡层。
颜色饱和度中等——比底层亮，但比第三层技能特效弱。
作用：让玩家不看HUD就知道当前天气。

烈阳天：暖色热雾 + 裂地橙红光
雷暴天：冷色雨丝 + 积水蓝白反光 + 闪电白闪
灵雨天：灰蓝雨丝 + 蓝色灵光粒子 + 积水涟漪
霜雪天：蓝白雪花 + 银白冰面反光 + 白雾
妖风天：黄褐沙尘带 + 风线弧形粒子 + 碎屑
晴    天：极淡银蓝月光 + 偶尔萤火微光
```

### 第三层高饱和粒子——战斗爆发专属

```
这层只在战斗动作发生时出现，是画面中色彩最浓烈的部分。

命中反馈：
  普通命中：小白色爆点，0.1秒消散
  重击/暴击：属性色大爆点（火橙/雷蓝/冰银/毒绿），0.3秒消散
  破势（状态引爆/地形连锁）：属性色爆点 + 屏幕边缘属性色光晕 0.5秒

万法归一释放：
  第0帧：全屏白色闪光（100%亮度，0.15秒）
  第0.15帧：属性色从角色位置向外扩散的冲击环
  焚天：橙红→金黄渐变火环
  雷罚：电蓝→白雷链
  沧海：冰蓝→银白冰波
  万毒：暗绿→紫毒雾
  不动：土黄→金色山岳虚影
  敌人死亡时：属性色粒子从敌人身体向四周飞溅

道统觉醒：
  屏幕边缘出现属性色纹路蔓延
  角色身上出现属性色光环
  道统名字：毛笔体，属性色渐变

连击里程碑：
  10连：金色小字飘出
  100连：金字+震屏+敌人慢放
  200连：全屏灰白化+角色唯一有颜色+金色光环

隐藏连锁触发：
  屏幕右下角弹出金色✦标签
  连锁效果本身用属性色粒子表现
  （如"雷引爆"：电弧从爆炸中心向外扩散，地面残留导电蓝光3秒）

完美闪避：
  角色原位残留白色残影0.05秒
  屏幕边缘白色闪光
  拖尾线：从原位到新位的白色光痕
```

## 二、各天气场景详细描述

### 天气一：烈阳

```
【第一层 · 水墨底】
  天空：顶部深棕黑，向下渐变为暗红棕
  地面：深灰近黑石板，水墨笔触纹理
  障碍物：浓墨剪影的碎石和残柱，边缘有飞白
  画面边缘：墨色晕染，微微带红调

【第二层 · 天象层】
  裂地：
    - 石板裂缝中透出暗橙红到亮橙红的光
    - 裂缝线条不规则，像干旱大地的龟裂
    - 裂缝密集区域形成明显的"裂地带"
    - 裂缝边缘有极细的热浪扭曲（空气折射）
  热雾：
    - 画面四角有橙色热雾缓慢翻涌
    - 饱和度中等——不是鲜艳的橙色，而是被水墨底压暗的暖橙
  地面碎石：
    - 散落的小碎石边缘被裂缝红光照亮
    - 碎石本身仍是墨色剪影，只有朝向裂缝的边缘有红光勾勒

【第三层 · 战斗爆发层】（战斗时才出现）
  火系技能命中：橙红→金黄爆点，火星四溅
  爆燃连锁：橙红火环从引爆点向外扩散，经过的裂地全部亮起
  焚天万法归一：金色火环从角色向外扩张，敌人化为灰烬飘散

【一眼可辨特征】
  红光裂纹 + 热浪扭曲 + 暗红棕天空
  整体暖色调，"地面在燃烧"
```

---

### 天气二：雷暴

```
【第一层 · 水墨底】
  天空：顶部极深靛黑，向下渐变为深靛蓝
  地面：深灰近黑石板，比其他天气更"湿"——表面有微弱反光
  障碍物：浓墨剪影，表面被雨水打湿后颜色更深
  画面边缘：墨色晕染，带冷蓝调

【第二层 · 天象层】
  积水：
    - 多片大小不一的积水区域
    - 积水呈深蓝黑色，表面有镜面般的微弱反光
    - 积水边缘有极细的电蓝光晕——暗示"导电"属性
    - 大水塘覆盖约1/4画面，小水洼约角色大小
  雨丝：
    - 极细、极密的白色雨丝从画面上方斜向下飘落
    - 雨丝本身是半透明白色，饱和度低
    - 落入积水时泛起小涟漪
  闪电：
    - 每3-5秒一道极亮白蓝色闪电撕裂天空
    - 闪电照亮整个画面0.1秒（全屏白闪）
    - 闪电消失后，积水表面瞬间闪亮一下，然后恢复暗蓝

【第三层 · 战斗爆发层】（战斗时才出现）
  雷系技能命中：电蓝→白爆点，电弧跳跃
  导电连锁：电弧从积水区的敌人跳向水中其他敌人，蓝白电弧在水面跳跃
  天雷劈下：金色→白的粗雷柱从天空劈向地面，落点周围积水全部电弧扩散
  雷罚万法归一：天雷连续劈下3-5道，每道锁定一个敌人，积水区域全部蓝白闪

【一眼可辨特征】
  积水蓝黑反光 + 闪电白闪 + 密雨丝
  整体冷色调，"天地在放电"
```

### 天气三：灵雨

```
【第一层 · 水墨底】
  天空：顶部深灰蓝，向下渐变为铅灰
  地面：深灰近黑石板，表面有一层薄薄的水膜反光
  障碍物：浓墨剪影的石头和残柱，表面湿润颜色深
  画面边缘：墨色晕染，灰蓝调

【第二层 · 天象层】
  薄积水：
    - 不是雷暴天的深积水塘，而是大面积薄薄的水膜
    - 低洼处形成浅积水区域，呈灰蓝色，微弱发光
    - 水面上密集的涟漪圆圈——雨滴不断落下
  灵雨粒子：
    - 比普通雨丝更粗、更慢的蓝色光点从空中缓缓飘落
    - 这是"灵气"——不是普通雨水
    - 灵光粒子落入积水后泛起蓝色光圈，比普通涟漪亮
  灵雾：
    - 地面有薄薄一层灰蓝色雾气，缓慢流动
    - 雾气密度不均匀，有些地方更浓，有些地方更淡
    - 比雷暴天更安静、更柔和

【第三层 · 战斗爆发层】（战斗时才出现）
  冻结连锁：冰蓝色从水面扩散，积水区域变成冰面
  冰裂：冰面碎裂时银白碎片飞溅
  毒雾扩散：暗绿色从毒杀点向外扩散，水面上毒雾更浓
  沧海万法归一：冰蓝水波从角色脚下扩散，经过区域全部冻结银白，然后冰裂爆开

【一眼可辨特征】
  大面积薄水膜 + 蓝色灵光粒子飘落 + 涟漪密集
  整体灰蓝冷调但比雷暴柔和，"天地灵气在流动"
```

### 天气四：霜雪

```
【第一层 · 水墨底】
  天空：顶部深暗蓝，向下渐变为灰白
  地面：深灰近黑石板，部分覆盖不均匀的霜
  障碍物：浓墨剪影，边缘挂着极简线条的冰凌
  画面边缘：墨色晕染，偏冷白调

【第二层 · 天象层】
  冰面：
    - 部分地面结了薄冰，呈高反光的银白色
    - 冰面反射周围暗色环境，像碎裂的镜面
    - 冰面上有细密的裂纹纹理
    - 冰面和未结冰的深灰地面形成强烈的明暗对比
  霜：
    - 未结冰的区域有蓝白色霜覆盖
    - 霜的颜色从深灰底中透出来，不是纯白
    - 覆盖不均匀，有些地方厚（更白），有些地方薄（露出深灰底）
  雪花：
    - 密集的蓝白色雪花缓慢飘落
    - 雪花不是纯白，微微发蓝光——"灵雪"
    - 比灵雨的粒子更大、更慢
  白雾：
    - 冰面裂缝中冒出白雾，缓慢上升后消散
    - 整体有一层薄薄的冷白雾气

【第三层 · 战斗爆发层】（战斗时才出现）
  冰系命中：冰蓝→银白爆点，冰晶碎片飞溅
  冻结：敌人身上冰从脚部向上蔓延覆盖，完全冻住后呈银白色
  冰裂：冻结敌人被击碎时，银白冰晶碎片向四周爆散，反射光线
  沧海万法归一：冰蓝水波扩散→冻结→冰裂连锁，全屏银白碎片

【一眼可辨特征】
  银白高反光冰面 + 深灰底 + 蓝白雪花 + 冰面白雾
  明暗对比极强，"天地在冻结"
```

### 天气五：妖风

```
【第一层 · 水墨底】
  天空：顶部深黄褐，向下渐变为灰黄
  地面：深灰近黑石板，表面覆盖沙尘和碎叶
  障碍物：浓墨剪影，迎风面有沙尘堆积
  画面边缘：墨色晕染，带黄褐调

【第二层 · 天象层】
  风线：
    - 由沙尘粒子组成的弧形线，从画面一侧吹向另一侧
    - 风线不是直线，是微微弯曲的弧线
    - 风线上粒子密度高，颜色偏黄褐
    - 风线外相对干净，形成清晰的"风路"
  风眼：
    - 画面中有一个明显的圆形平静区
    - 风眼中心几乎无尘，能看清地面水墨纹理
    - 周围风力最强，沙尘粒子密度最高
    - 风眼会缓慢移动（房间势：风眼游移）
  沙尘粒子：
    - 从画面左侧向右侧高速移动（或随风眼旋转）
    - 粒子大小不一——大的是碎叶碎纸，小的是沙尘
    - 整体呈黄褐色，饱和度中等
  远处尘墙：
    - 画面远端有一面黄褐色的尘墙
    - 尘墙边缘模糊，像水墨晕染

【第三层 · 战斗爆发层】（战斗时才出现）
  火系穿过风眼：橙红弹道变成螺旋火焰（火焰龙卷）
  弹幕被风改变方向：子弹/弹道的轨迹随风线弯曲
  万法归一在风中：效果沿风线扩散更远

【一眼可辨特征】
  黄褐沙尘带 + 风线弧形粒子 + 风眼平静区 + 碎屑飘飞
  整体混沌偏暖黄，"天地在呼啸"
```

### 天气六：晴（教学/新手天气）

```
【第一层 · 水墨底】
  天空：顶部极深靛黑，向下渐变为深靛蓝
  地面：深灰近黑石板，干净干燥，纹理清晰
  障碍物：浓墨剪影，轮廓最清晰——没有天气效果遮挡
  画面边缘：墨色晕染，中性冷调

【第二层 · 天象层】
  月光：
    - 画面顶部有极淡的银蓝色月光，从上往下渐弱
    - 月光给场景顶部一层微弱的冷色调提亮
  萤火微光：
    - 偶尔一两只极小的银蓝色微光粒子缓慢飘过
    - 粒子极小、极慢、极淡——不是主角，只是点缀
  地面状态：
    - 无积水、无冰面、无裂地、无毒雾
    - 地面是"安全的"——没有任何地形效果
  空气：
    - 几乎无粒子效果，画面最"干净"
    - 空气清澈，远近障碍物都能看清

【第三层 · 战斗爆发层】
  晴天没有天象加成，技能特效纯靠自身属性色
  这反而让战斗特效在干净的画面上最醒目
  完美闪避的白色残影在无干扰环境下最清晰

【一眼可辨特征】
  干净地面 + 极淡月光 + 无天气粒子
  画面信息密度最低，"天地在等你出手"
```

## 四、AI 生图统一前缀

所有天气的 AI 生图提示词都应包含以下画风前缀：

```
[画风前缀——所有天气通用]
45-degree isometric top-down view of a dark Chinese cultivation game 
battle scene. Dark ink-wash painting aesthetic as the base layer — 
deep grey-black stone floor with subtle ink brush texture and paper 
grain, ink-black silhouette obstacles with rough brushstroke edges, 
deep gradient sky with ink-wash blending at edges like rice paper 
bleeding. Extremely low saturation on the base scene — almost 
grayscale, relying only on light and dark contrast. All vivid colors 
come from weather effects and combat particle effects layered on top. 
The player character is the brightest element in the scene — small 
figure with a thin white glow outline and a luminous sword, standing 
out sharply against the dark ink-wash background. Enemies are dark 
silhouettes with subtle weapon outlines. Minimal semi-transparent UI 
elements. Overall mood: dark, atmospheric, with explosive bursts of 
high-saturation color during combat moments.
```

在前缀之后，加上对应天气的特有描述即可。
