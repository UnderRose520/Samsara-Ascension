# 《轮回仙途》地图尺寸、布局与角色规格——64×64 基准版

> 基准单位：玩家角色碰撞体 64×64 像素
> 画风：暗色水墨底 × 高饱和粒子特效
> 视角：45度等距俯视角

---

## 一、基准单位定义

### 1.1 角色与怪物的像素基准

```
玩家角色：
  碰撞体：64×64 像素
  视觉体：72×72 像素（碰撞体 + 外围轮廓光和飞白毛边）
  
  角色身体（剪影部分）：约 48×56 像素
    头部：约 14×14 像素
    躯干：约 20×24 像素
    双腿：约 16×18 像素
    角色在 64×64 碰撞框内居中，上下左右各留约 4-8 像素空间
    
  轮廓光：1-2 像素宽，沿身体剪影内侧环绕
  飞白毛边：轮廓线外 1 像素随机散点
  道途标识光效：在角色身体的特定位置，额外占 8-20 像素

怪物体型参照表（以 64×64 角色为 1.0 基准）：

  ┌──────────────┬──────────┬──────────┬──────────┐
```

### 1.2 地面瓦片/网格基准

```
不使用瓦片地图，但用网格做逻辑参考：

逻辑网格单元：64×64 像素（和角色碰撞体同尺寸）
这意味着角色刚好占 1 个格子

网格只用于：
  - 碰撞检测的空间分区
  - 障碍物放置的位置参考
  - 天象地形区域的面积计算
  - 敌人出生点的分布参考

网格不用于：
  - 视觉渲染（地面不是方格拼接）
  - 移动限制（角色自由移动，不受网格对齐限制）
  - 地形形状（积水/裂地/冰面是自由形状，不是网格填充）
```

## 二、战斗房间尺寸

### 2.1 各房间类型尺寸（以 64px 角色为基准）

```
普通战斗房：
  尺寸：1920×1920 像素
  等效网格：30×30 格
  角色从中心走到边缘：约 15 格 = 约 2.5 秒（中速移动 190px/s）
  可视区域（1920×1080 屏幕）约为房间的 56%×56%
  玩家在屏幕中央时，房间四角刚好在屏幕边缘外
  → 需要轻微镜头跟随/平移才能看到整个房间

精英战斗房：
  尺寸：2112×2112 像素
  等效网格：33×33 格
  比普通房大约 10%
  给精英的招式和走位留更多空间

事件房：
  尺寸：1280×1280 像素
  等效网格：20×20 格
  事件房不需要大面积战斗空间
  聚焦在交互元素上

坊市/调息房：
  尺寸：960×960 像素
  等效网格：15×15 格
  纯 UI 交互，空间紧凑

Boss 房：
  尺寸：2560×2560 像素
  等效网格：40×40 格
  Boss 体型约 160×160（2.5 倍角色），需要大舞台
  Boss 冲锋/砸地等大范围招式需要足够展开空间

最终 Boss 房（渡劫）：
  尺寸：2880×2880 像素
  等效网格：45×45 格
  终局最大舞台
  天劫招式覆盖面积最大
```

```
│  │  └────┘                          └────┘      │    │
│  │          ┌──────────────┐                     │    │
│  │          │  天象地形区   │                     │    │
│  │          │ （积水/裂地） │        ○ 玩家出生   │    │
│  │          │              │                     │    │
│  │          └──────────────┘                     │    │
│  │    敌人出生区               敌人出生区         │    │
│  │         ★                       ★            │    │
│  │                              ★                │    │
```

#### 边界墙区

```
宽度：房间四边各 64-96 像素（1-1.5 格）
视觉：浓墨色墙壁/悬崖/迷雾
  - 比障碍物更深的墨色（近乎纯黑 RGB 10-15, 8-12, 10-15）
  - 顶部边缘有墨色晕染向房间内扩散 16-32 像素
  - 像宣纸边缘的自然洇开效果
  - 墙壁表面无纹理——纯墨色块
  
碰撞：不可穿越
  角色和敌人碰到边界墙会被阻挡
  弹幕碰到边界墙会消散
  某些技能可以利用边界墙反弹（如弹幕手的弹幕、风系弹道）

不可放置区域：
  边界墙区内不放置任何敌人出生点、天象地形、特征物
  确保玩家不会被逼到墙角无法脱身
```

#### 障碍物区

```
面积占比：房间的 15-25%（约 25-40 个网格单元）
障碍物类型和尺寸：

  石堆（小）：
    碰撞体：64×64（1×1 格）
    视觉：浓墨色不规则石堆剪影
    可被强技能破坏（如爆燃、万法归一）
    破坏后碎成 3-4 块小碎片飞散

  石堆（中）：
    碰撞体：128×96（2×1.5 格）
    视觉：两三个石块堆叠的剪影
    不可破坏
    高度视觉：石堆下方有淡墨色"阴影"（8 像素宽）

  残柱：
    碰撞体：64×64（1×1 格）
    视觉：垂直的矩形剪影，顶部有断裂的不规则边缘
    柱身上有极淡的墨色纹理（模拟石柱表面）
    不可破坏

  断墙（长）：
    碰撞体：192×64（3×1 格）
    视觉：横向的长条矩形剪影
    墙面有 1-2 处断裂口（视觉上可穿过，但碰撞体是完整的）
    断墙可以阻挡弹道——是战术掩体

  断墙（L形）：
    碰撞体：192×64 + 64×128（L形组合）
    视觉：L 形的墙壁剪影
    形成一个角落——适合聚怪后引爆
    是最有战术价值的障碍物形状

  废墟团：
    碰撞体：128×128（2×2 格）
    视觉：多块碎石和断壁的不规则组合
    最大的单体障碍物
    可以在废墟两侧形成"通道"

障碍物放置规则：
  1. 任意两个障碍物之间至少留 128 像素（2 格）间距
     → 确保角色（64px）可以穿过
  2. 不允许形成死胡同——从任意可通行点到另一点至少有 2 条路径
  3. 障碍物不能放在房间中央的 256×256 区域内
     → 中央是主要战斗区，必须开阔
  4. 至少有 2 处"L形角落"或"通道"
     → 给聚怪→引爆的战术链提供地形支撑
  5. 障碍物群形成 2-3 个"岛"——不是散落均匀分布
     → 让空间有"区域感"
```

#### 天象地形区

```
每种天气有 1-3 片天象地形区域，每片有最小面积要求：

烈阳裂地：
  数量：2-3 片
  每片面积：至少 192×192 像素（3×3 格）
  总覆盖面积：房间的 15-20%
  形状：不规则的裂纹网
    主裂纹线：从中心向四周放射，宽 4-8 像素
    次裂纹线：从主裂纹分支，宽 2-4 像素
    裂纹密度：中心密、边缘疏
  视觉：
    裂纹本身发出暗橙红→亮橙红的光
    裂纹周围 16 像素内有微弱的热浪扭曲
    裂纹区域的地面纹理比周围略亮（被红光照射）

雷暴积水：
  数量：2-4 片
  每片面积：至少 128×128 像素（2×2 格）
  其中至少 1 片面积 ≥ 256×256（4×4 格）
  总覆盖面积：房间的 18-25%
  形状：不规则的水洼
    边缘不规则——有凹凸和小半岛
    水洼之间可以有窄水道连接（8-16 像素宽）
  视觉：
    积水表面比周围地面亮 2-3 个等级
    积水呈深蓝黑色，有镜面微反光
    积水边缘有极细的电蓝光晕（2 像素宽）
    雨滴落入时泛起涟漪（圆圈动画，直径 8-16 像素，0.5 秒消散）

灵雨薄积水：
  数量：整片覆盖
  覆盖面积：房间的 40-50%（大面积薄水膜）
  形状：覆盖大部分地面，但高处（障碍物顶部）没有
    不需要精确形状——整个地面都有一层薄薄的水膜
    低洼处水更深（颜色更亮、涟漪更密）
  视觉：
    整体地面比晴天亮 1 个等级
    低洼处有灰蓝色反光
    雨滴涟漪密集覆盖全地面

霜雪冰面：
  数量：2-3 片
  每片面积：至少 192×192 像素（3×3 格）
  总覆盖面积：房间的 12-18%
  形状：不规则的冰面区域
    边缘有霜的渐变过渡（从冰面→霜→正常地面）
    冰面上有裂纹纹理线（宽 1-2 像素）
  视觉：
    冰面呈高反光银白色——比周围地面亮 4-5 个等级
    冰面反射周围暗色环境（用简单的颜色映射，不是真反射）
    裂纹纹理线比冰面略暗
    冰面边缘有白雾粒子上升（3-5 个/秒）

妖风风线：
  数量：3-5 条风线 + 1 个风眼
  风线宽度：64-128 像素（1-2 格宽）
  风线长度：贯穿房间的一个方向（约 1600-1900 像素）
  风眼直径：128-192 像素（2-3 格）
  形状：
    风线是弧形的——从房间一侧吹向另一侧
    风眼是圆形的平静区，风线围绕风眼弯曲
  视觉：
    风线内有密集的沙尘粒子（黄褐色，2-3 像素大小）
    粒子沿风线方向高速移动（200-300px/s）
    风线边缘粒子密度递减
    风眼中心几乎无粒子——能看到干净的地面
    风眼边缘粒子密度最高

晴天：
  无特殊地形
  地面干净、干燥
  是"教学天气"——没有任何地形干扰
```

#### 敌人出生区

```
出生区域：分布在房间的外环区域（距中心 320-768 像素）
  不在中央战斗区出生（避免和玩家零距离）
  不在边界墙区内出生（避免被卡住）
  不在障碍物内部出生

出生密度控制：
  普通战斗房：第一波 4-6 个敌人
  后续每波 3-5 个
  同屏最多 12-15 个活跃敌人
  Boss 房小怪波：每波 3-4 个

出生点间距：
  任意两个敌人出生点之间至少 128 像素（2 格）
  避免敌人重叠出生

不同敌人的出生位置偏好：
  追猎者（近战）：距离玩家 256-512 像素，开阔区域
  弹幕手（远程）：距离玩家 512-768 像素，障碍物后方
  护阵者（辅助）：和其他敌人一起出生，不单独
  自爆魔（冲锋）：距离玩家 384-640 像素，直线路径上无障碍
  召唤者（远程）：距离玩家 640-768 像素，最远
  狙击者（远程）：距离玩家 640-768 像素，障碍物掩体后
```

#### 特征物区

```
每房 0-1 个特征物，放在房间偏僻角落（距中心 ≥ 512 像素）

灵泉：
  碰撞体：无（可穿过）
  视觉体：48×48 像素的发光圆
  颜色：蓝白色，向外扩散水纹圈（每 1.5 秒一圈）
  效果：站在上面每秒回复 3% 生命
  交互：不需要按键——踩上去自动生效

矿脉：
  碰撞体：64×32（不可穿过）
  视觉体：64×40 的岩石剪影，表面有灵石光点
  颜色：岩石深墨色，灵石点呈淡金色
  效果：攻击矿脉爆出灵石（5-8 次）
  交互：需要攻击

古阵：
  碰撞体：无
  视觉体：96×96 的地面法阵图案
  颜色：暗金色线条，缓慢旋转
  效果：触发随机事件（好的或坏的）
  交互：踩上去触发

石碑：
  碰撞体：32×64（不可穿过）
  视觉体：32×72 的竖立碑石剪影
  颜色：深墨色，碑面有淡金色文字线条
  效果：提供修仙知识/功法线索/隐藏连锁提示
  交互：靠近后自动显示文字
```

```
    → 不需要镜头平移——角色固定在屏幕中央

    玩家看不到房间的上下边缘
    但敌人从屏幕外走入场内（有提前预警音效）
```
