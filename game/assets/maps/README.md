# Map assets (UIUX В§9 stage themes)

Procedural tilesets and combat backgrounds generated by `game/tools/generate_2d_maps.py`.

## Regenerate

```bash
python game/tools/generate_2d_maps.py
```

Requires [Pillow](https://pypi.org/project/Pillow/): `pip install Pillow`

## Layout

Each theme folder contains:

| File | Size | Purpose |
|------|------|---------|
| `tileset.png` | 128Г—128 | Four 32Г—32 tiles: floor, floor_alt, wall, decoration |
| `room_background.png` | 1280Г—720 | Subtle gradient + pattern for combat arena |

`tileset_metadata.json` lists `tile_size` (32), `columns` (4), and theme ids aligned with `data/stages/stages.csv`.

## Combat floor scene

**Scene:** `res://scenes/rooms/combat_floor.tscn`

Instance as a child of `RunController` (`res://scenes/rooms/run_controller.tscn`):

```gdscript
var floor_scene := preload("res://scenes/rooms/combat_floor.tscn")
var floor := floor_scene.instantiate()
add_child(floor)
floor.apply_theme(RunContext.current_stage + 1)  # stage_index 1вҖ“5
```

Structure:

- `Background` (`Sprite2D`) вҖ” full 1280Г—720 backdrop, anchored top-left at (-640, -360)
- `TileMap` / `FloorLayer` (`TileMapLayer`) вҖ” checkerboard floor using atlas tiles 0/1

Call `apply_theme(stage_index)` after instancing to swap textures from `tileset_metadata.json`.
